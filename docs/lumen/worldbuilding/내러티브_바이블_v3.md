# 루멘 내러티브 바이블 v3

> 신들은 영원하지만 완벽하지 않다. 딜러들은 유한하지만 세계를 바꾼다.
> 해글은 도달할 수 없지만, 가까워지는 모든 순간이 이야기가 된다.

**세계관 기반**: [루멘 세계관 v4](./루멘_세계관_v4.md)

**v2와의 차이**: 세계관이 "평행 세계 + 현실 연결" 구조로 바뀜에 따라, 유저의 현실 경험과 스토리가 직접 닿는 부분을 명시. 전체 서사 철학은 유지.

---

## 1. 내러티브 원칙

### 하지 않는 것

- **선악 이분법 없음** — 적대 세력의 주장에 일리가 있어야 한다. 유저가 "저쪽 말도 맞는데..."라고 느끼면 성공
- **자기 복제 없음** — 같은 갈등 구조를 반복하지 않는다. 시즌마다 새로운 종류의 질문을 던진다
- **전부 다 풀지 않음** — 미스터리는 유지. 해글이 뭔지, 빛이 왜 나는지 — 답을 주면 이야기가 끝난다
- **주인공 보정 없음** — 영웅도 틀리고, 악인도 맞고, 신도 실수한다
- **유저를 캐논의 주인공으로 만들지 않음** — 유저는 자기 거래로 루멘에 기여하지만, 캐논의 사건은 캐논 캐릭터들의 것이다. "당신이 선택된 자"는 금지

### 하는 것

- **매 시즌 새로운 질문** — 이전 시즌의 답이 다음 시즌의 질문이 된다
- **정통적 흐름** — 발단→위기→절정→변화. 구조는 클래식하게, 내용은 신선하게
- **조용한 장면이 가장 강하다** — 전투가 아니라 선택의 순간이 클라이맥스
- **유저의 거래가 이야기** — 유저의 모든 거래가 루멘의 빛을 만드는 *실제* 행위라는 감각. 메타포가 아니라 세계의 물리법칙
- **현실과 루멘의 이중 언어** — 같은 사건을 현실 관점과 루멘 관점 두 층으로 그린다. 일반인은 반딧불이로 보고, 딜러는 피즈(버디)로 본다

## 2. 캐릭터 계층

```
┌─────────────────────────────────┐
│         원시 존재 (6체)           │  영원. 변하지 않는다 — 하지만 깨달을 수 있다
├─────────────────────────────────┤
│       적대 세력 보스 (4인)        │  이데올로그. 각자의 정의를 가진 자들
├─────────────────────────────────┤
│       주역 딜러 (영웅들)          │  성장. 선택. 세계를 움직이는 유한한 존재들
├─────────────────────────────────┤
│       일반 딜러 (유저 투영)        │  유저 자신의 거울. 보통 사람들의 이야기
└─────────────────────────────────┘
```

### 원시 존재 — 신들의 관계도

신들은 변하지 않는다. 하지만 세계를 통해 **자기 자신을 이해해간다.** 아바타 형태로 현실에 내려와 인간들 사이를 걷는 것이 그 방법 중 하나다.

**핵심 관계 3개** (전 시즌 관통):

| 관계 | 긴장 | 해소 방향 |
|------|------|----------|
| 포르자 ↔ 볼라 | 창조자는 무관심하고, 욕망은 채워지지 않는다 | "포르자가 뒤를 돌아보는 순간" = 해글의 가장 깊은 힌트 |
| 유스 ↔ 클레멘 | 둘 다 맞지만 양립 불가 | 해소되지 않는다 — 그게 공정의 본질 |
| 리브라 ↔ 테소르 | 잴 수 없는 것 vs 놓을 수 없는 것 | 리브라의 첫 감정, 테소르의 첫 놓아줌 |

### 적대 세력 — 4개의 논리

**각 세력은 한 단어로 요약된다:**

| 세력 | 리더 | 한 단어 | 프로토콜을 향한 지적 |
|------|------|--------|-------------------|
| 다크 미들러 | 메르카 | **착취** | "프로토콜도 수수료 받는 중개자 아닌가?" |
| 이퀄리스트 | 파리아 | **불평등** | "공정한 기회가 공정한 결과를 보장하나?" |
| 고요파 | 고요의 자 | **번아웃** | "빛을 만드는 체계가 사람을 소모하고 있다" |
| 순수파 | 프리마 | **타락** | "동반자를 상품화하는 순간 가장 아름다운 것이 죽는다" |

**세력 간 역학** (시즌이 진행될수록 동맹/분열이 변한다):
```
다크 미들러 ←→ 이퀄리스트     정반대. 불평등에서 이익 vs 불평등 해소
   ↕                ↕
순수파     ←→    고요파        "더 순수하게 거래" vs "거래를 멈추자"
   ↑                           
   └── 다크 미들러가 뒤에서 지원할 동기 (프로토콜 해체 시 중개자 다시 필요)
```

**세력의 깊이 — 역사적 층위**: 각 세력은 루멘 고유가 아니라 인류 역사에서 반복되어 온 힘들의 집단 기억이 자아를 얻은 것. 메르카는 한 명의 악인이 아니라 수천 년 된 계보의 현재 대변자. 이 층위는 시즌 4~6에서 점점 드러난다. 시즌 1~3에서는 암시만.

### 주역 딜러 — 씨앗만 심는다

구체적 캐릭터는 시즌별로 개발. 아래는 **역할 유형**만 정의:

| 역할 | 기능 | 예시 방향 |
|------|------|----------|
| **탐구자** | 해글이 뭔지 묻는 사람. 메인 퀘스트 | 평범한 딜러가 첫 거래에서 각성 |
| **심판자** | 유스/클레멘의 갈등을 인간 레벨에서 재현 | 배심원이 된 딜러. 가장 어려운 판정 |
| **증인** | 과거를 직접 아는 유일한 사람 | 결빙에서 깨어난 고대 딜러 |
| **이방인** | 시스템 밖에서 온 존재. 기존 질서에 질문 | 앱 없이 루멘을 볼 수 있는 자. 태그 불명 버디 보유자 |

## 3. 매크로 아크 — 전체 흐름

### 3막 구조 (시즌 단위)

```
제1막: 빛의 시대                    제2막: 그림자의 시대               제3막: 선택의 시대
(시즌 1~2)                         (시즌 3~5)                        (시즌 6~)
                                                                    
"세계는 아름답다"                    "세계는 아름답지만 아프다"          "그래서 어떻게 할 것인가"
                                                                    
프로토콜 활성화                      적대 세력 부상                     세력 간 동맹/분열
막이 얇아짐                         미스터리 심화                      원시 존재들의 변화
지역이 보이기 시작                   기존 답의 균열                     유저가 방향을 선택
                                                                    
질문: "빛은 왜 나는가?"              질문: "빛이면 충분한가?"            질문: "해글은 가능한가?"
```

### 시즌별 테마 — 이중 질문 구조

각 시즌은 두 개의 질문을 가진다:
- **표면 질문**: 루멘 세계 안의 사건에 대한 질문. 스토리를 진행시킨다.
- **이면 질문**: 유저의 현실 삶에 닿는 철학적 질문. 답을 주지 않는다.

표면 질문은 시즌 안에서 (불완전하게) 풀린다. 이면 질문은 **절대 풀지 않는다.**
유저가 거래하다가, 버디를 키우다가, 분쟁을 겪다가 — 문득 이면 질문이 스치면 성공.

| 시즌 | 표면 질문 (세계) | 이면 질문 (삶) | 엔딩 |
|------|---------------|-------------|------|
| 1 | "빛은 왜 나는가?" | **내가 하는 일에 의미가 있는 건 어떻게 아는가?** 매일 출근하고 일하고 돌아온다. 규칙을 지킨다. 근데 그게 의미 있는 건지는 누가 알려주나. 포르자는 만들기만 하고 뒤를 안 본다. 우리도 그런 건 아닌가. | 규칙을 다 지켰는데 빛이 안 나옴 |
| 2 | "탁한 빛도 빛인가?" | **불완전한 선의도 선의인가?** 완벽하게 좋은 의도로만 행동하는 게 가능한가. 메르카는 세계를 살렸지만 대가를 원했다. 순수하지 않은 동기에서 나온 좋은 결과는 좋은 것인가. | 고요의 자 첫 등장 |
| 3 | "멈추는 것이 답인가?" | **쉬는 것과 포기하는 것의 차이는 무엇인가?** 번아웃은 진짜다. 쉬어야 할 때 쉬는 것은 지혜다. 근데 "영원히 쉬는 것"은 지혜인가 포기인가. 고요의 자는 위로하는 건가, 죽이는 건가. 내가 지금 쉬고 있는 건 충전인가 회피인가. | 예상 못한 선택 |
| 4 | "공정한 판정이 가능한가?" | **내가 옳다는 확신은 어디서 오는가?** 유스는 확신한다. 클레멘도 확신한다. 둘 다 맞다. 내가 누군가를 판단할 때, 그 확신은 증거에서 오는 건가 편견에서 오는 건가. 중립이 가능하긴 한가. | 원시 존재 하나가 변함 |
| 5 | "과거가 정말 더 좋았는가?" | **향수는 기억인가 창작인가?** "옛날이 좋았다"는 말은 사실에 기반한 건가, 현재에 대한 불만을 과거에 투사한 건가. 프리마가 본 최초의 빛은 정말 더 순수했나, 단순했기 때문에 순수해 보인 건가. | 미공개 흐름 각성 |
| 6+ | 이전 시즌에서 도출 | — | — |

**핵심 규칙**:
- 시즌 N의 표면 질문은 시즌 N-1의 답에서 나온다. 미리 정하지 않는다.
- 이면 질문은 **대사로 직접 말하지 않는다.** 상황과 캐릭터의 행동으로 유저가 스스로 도달하게 한다.
- "이 시즌의 교훈은 X입니다" 같은 정리는 절대 하지 않는다.

### 이면 질문이 작동하는 법

직접 묻지 않는다. 상황이 묻는다.

```
❌ 이렇게 하지 않는다:
   캐릭터 A: "쉬는 것과 포기의 차이가 뭘까?"
   캐릭터 B: "그건 의도의 차이야. 돌아올 마음이 있으면 쉬는 거고..."

✅ 이렇게 한다:
   지친 딜러가 거래를 멈춘다. 버디가 잠든다.
   3개월 후 다시 거래를 시작한다. 버디가 깨어난다.
   옆에서 같이 멈췄던 딜러는 돌아오지 않았다.
   아무도 "왜"를 설명하지 않는다.
   유저가 스스로 생각한다.
```

### 전체 관통 질문 — 해글의 질문

모든 시즌을 관통하는 하나의 큰 질문이 있다. 이것은 시즌별 질문 위에 있는 상위 질문:

> **"완벽하지 않은 것이 완벽한 것보다 나을 수 있는가?"**

프로토콜은 완벽하지 않다. 원시 존재들도 스스로 인정한다. 하지만 타락한 존재의 통제된 세계는 완벽하다 — 아무도 울지 않는다. 대신 아무도 웃지도 않는다.

불완전한 자유 vs 완벽한 통제. 아픔이 있는 삶 vs 아픔이 없는 무. 이것은 루멘의 질문인 동시에 삶의 질문이다.

답을 주지 않는다. 유저가 거래하고, 실패하고, 성공하고, 분쟁하고, 화해하는 경험 자체가 답의 재료가 된다.

## 4. 스토리 엔진 — 반복하지 않는 법

### 갈등 유형 로테이션

같은 갈등을 반복하면 자기 복제가 된다. 시즌마다 **갈등의 종류 자체를 바꾼다:**

```
시즌 1: 신비 (미스터리) — "이게 뭐지?"
시즌 2: 도덕 (윤리) — "이게 옳은 건가?"  
시즌 3: 실존 (존재) — "왜 해야 하는가?"
시즌 4: 판정 (정의) — "누가 결정하는가?"
시즌 5: 시간 (기억) — "정말 그랬는가?"
```

### 서프라이즈 원칙

매 시즌 최소 하나의 **"이건 예상 못 했다"**:

- 악인이라 생각한 캐릭터가 옳았던 순간
- 영웅이 명백하게 틀린 순간
- 원시 존재가 예상 밖의 행동을 하는 순간
- 유저가 감정이입한 세력이 흔들리는 순간

**절대 하지 않는 서프라이즈**: 배신 트위스트 남용, 사망 복구, 기억 조작. 값싼 반전은 쓰지 않는다.

### 신선함의 원천

| 방법 | 설명 |
|------|------|
| **시점 전환** | 같은 사건을 다른 캐릭터 시점으로 다시 보여준다. 특히 현실 시점 vs 루멘 시점 |
| **규모 변경** | 신들의 갈등 → 개인의 일상으로 축소. 또는 그 반대 |
| **질문 전복** | "누가 악인인가?" → "악인이라는 개념이 유효한가?" |
| **침묵** | 가장 중요한 순간에 대사 없이 행동만으로 보여준다 |
| **미공개 흐름 각성** | 새 원시 존재가 깨어나면 세계의 규칙 자체가 확장된다 |
| **가시성 레이어 활용** ← 신규 | 일반인은 반딧불이로 보고, 딜러는 버디로 보는 이중 장면 |

## 5. 장기 씨앗 — 심어두고 나중에 쓸 것

지금은 언급만 하고 설명하지 않는다. 나중에 필요할 때 꺼낸다.

| 씨앗 | 심는 시점 | 쓰는 시점 | 내용 |
|------|---------|---------|------|
| 빛이 안 나오는 거래 | 시즌 1 엔딩 | 시즌 3+ | 규칙을 다 지켰는데 왜? |
| 리브라의 첫 망설임 | 세계관 기본 | 시즌 4+ | 최초의 분쟁에서 판단 못 한 것이 첫 감정이었나? |
| 포르자가 뒤를 돌아봄 | 시즌 1~4 암시 | 시즌 6+ | 해글의 핵심 열쇠 |
| 고요의 자의 기원 | 시즌 2 첫 등장 | 미정 | 언제부터 있었는가? 답을 정하지 않는다 |
| 측정 불가능한 존재 | 시즌 1~ | 장기 | 리브라의 척도가 안 먹히는 것이 왜 존재하는가? |
| 미공개 원시 흐름 (6+) | 세계관 부록 | 시즌별 1~2 | 새 흐름 = 새 규칙 = 세계 확장 |
| 타락한 존재 (이름 미정) | 미정 | 시즌 5+ | 전체 시즌을 관통하는 배후? 아닐 수도 |
| 역사적 세력의 층위 | 시즌 1~3 암시 | 시즌 4~6 | 4세력이 루멘 고유가 아니라 인류사의 그림자라는 사실 |
| 두 세계의 과거 버전 | 시즌 1 마르코 | 시즌 3~5 | 제3시대의 루멘, 결빙 전의 루멘 |

## 6. 톤 가이드

### 대사 톤

```
원시 존재:  짧다. 무겁다. 한 문장이 한 시즌을 관통한다.
           "쉬는 것과 멈추는 것은 다르다." — 클레멘

적대 보스:  논리적이다. 반박하기 어렵다. 감정이 아니라 이성으로 설득한다.
           "0의 100%보다 탁한 빛이라도 있는 게 낫지 않은가?" — 메르카

주역 딜러:  인간적이다. 유머가 있다. 완벽하지 않다.
           "Mythic 버디가 강한 건 알겠는데, 내 이 녀석은 그냥... 좋거든?"

일반 딜러:  일상적이다. 거창하지 않다. 유저의 언어.
           "해글? 모르겠고, 오늘 거래 잘 끝나서 기분 좋다."

일반인:     루멘을 못 보는 자. 버디를 반딧불이로, 그림자를 착시로 본다.
           "젊은이, 반딧불이 잡았어? 예쁘네."
```

### 감정 곡선

```
각 시즌의 감정 흐름:

호기심 → 몰입 → 불안 → 갈등 → [조용한 클라이맥스] → 여운
                                    ↑
                             여기가 가장 강하다
                             폭발이 아니라 선택
```

### 현실-루멘 이중 톤

새 세계관에서 장면을 그릴 때 항상 두 층을 고려한다:

```
현실 층: 카이가 스마트폰을 건네며 돈을 받는다. 손님이 웃는다. 평범한 거래.

루멘 층: 결속의 빛이 터진다. 피즈가 태어난다. 볼트랜드의 네온 하나가 더 밝아진다.

교차 지점: 카이만 둘 다 본다. 손님은 그냥 좋은 거래를 했다고 느낄 뿐이다.
```

대부분의 장면은 이 이중 구조를 **드러내지 않고** 쓴다. 가끔, 특별한 순간에만 둘을 겹쳐 보여준다. 그게 시그니처 장면이 된다.

## 부록: 금지 사항

| 절대 하지 않는 것 | 이유 |
|----------------|------|
| 해글의 정체를 밝힌다 | 이야기가 끝난다 |
| 이면 질문에 답을 준다 | 캐릭터가 "교훈"을 말하는 순간 철학이 설교가 된다 |
| 원시 존재가 죽는다 | 영원한 존재. 변할 수는 있지만 사라지지 않는다 |
| 악의 없는 적대 세력을 순수악으로 전환 | 세계관의 핵심인 양면성이 무너진다 |
| 같은 구조의 갈등을 반복 | "또 공의 vs 자비야?" → 죽은 서사 |
| 유저의 거래와 무관한 스토리 | 유저 활동이 세계를 만든다는 원칙 위반 |
| 사망 → 부활 트릭 | 값싼 감정 소비 |
| "선택된 자" 설정 | 평범한 자가 선택하는 것이 핵심 |
| 나레이션으로 주제를 설명 | "이 이야기가 말하고자 하는 것은..." → 즉시 죽는 서사 |
| 일반인을 바보로 그리기 ← 신규 | 일반인이 루멘을 못 보는 건 무능이 아니라 상태다. 조롱 대상이 아님 |
| 루멘을 "진짜"로, 현실을 "가짜"로 취급 ← 신규 | 두 세계는 동등하게 실재한다 |
