# 버디 도감 (Buddy Encyclopedia)

> *라텐스의 가능성이 공정한 거래의 빛을 받아 태어난 존재들.*
> *같은 종족이라도 딜러에 따라 다른 개체가 태어난다.*

**등급 체계**: Common → Uncommon → Rare → Epic → Legendary → Mythic → Awakened

**참조**: [루멘 세계관 v4](./루멘_세계관_v4.md) · [게이미피케이션 설계](./게이미피케이션_설계.md)

---

## 종족과 개체

이 도감은 **종족(Species)**을 기술한다. 캐논 캐릭터들의 버디는 종족에 속하는 **개체(Individual)**이며, 고유 이름을 가진다.

| 개체 이름 | 소유자 | 종족 | 비고 |
|----------|-------|------|------|
| 피즈 (Fizz) | 카이 | — (전구형, Spark) | Common 탄생. 감정 연동 발광 |
| 글림 (Glim) | 진 | — (태그 불명) | 시스템 분류 불가 |
| 프로스트 (Frost) | 마르코 | — (얼음 결정, Stone) | Legendary 탄생 |
| 넥서스 (Nexus) | 메르카 | — (빛의 네트워크, Prism) | Epic 탄생 |
| 에코 (Echo) | 벨 | — (투명 해파리, Tide) | Epic 탄생 |
| 시그니스 (Cygnis) | 팹 | — (금속 새, Forge) | Legendary 탄생 |
| 기타 | — | — | [시즌1 문서](./루멘_시즌1_깨어남_v2.md) §4 참조 |

> 캐논 개체의 종족 분류는 미확정. 일부는 기존 종족에 속하지 않는 고유 존재일 수 있음.

## 진화 시스템

버디는 태어난 뒤 딜러와 함께 거래하며 **성장(진화)**한다.
진화하면 외형이 변하고 현재 등급이 한 단계 상승한다.

```
탄생 등급 (에그 확률로 결정)  →  진화 천장 (일반 경로 최대)
Common / Uncommon           →  Epic까지
Rare / Epic                 →  Legendary까지
Legendary                   →  Mythic까지

각성 (Awakening): 천장 초월. 탄생 등급 무관. 극소수만.
```

아래 종족별 "진화 단계"는 버디가 BLv(버디 레벨)이 오르며 거치는 외형 변화를 기술한다.

## 등급별 공통 변화 규칙

| 등급 | 테두리 | 외형 변화 | 비고 |
|------|--------|----------|------|
| ⬜ Common | 흰색 | 기본 색상, 무늬 없음 | 가장 흔한 형태 |
| 🟩 Uncommon | 초록 | 채도 상승, 부분 발광 시작 | 눈이나 말단부에 미세한 빛 |
| 🟦 Rare | 파란 | 고유 무늬 출현 (줄무늬/별/물방울) | 종족 특성이 선명해짐 |
| 🟪 Epic | 보라 | 특수 색상 (홀로그램/네온/메탈릭) | 오라 미세하게 발생 |
| 🟨 Legendary | 금 | 전설 전용 디자인 + 능력 2개 | 주변 공기에 영향 |
| 🔴 Mythic | 홀로그램 | 종족당 1~2체 한정 + 능력 강화 | 루멘의 날씨가 반응 |
| 🌟 Awakened | — | 고유 이름, 문양, 멘토 전환 | 세계에 하나뿐인 존재 |

# ⚡ 볼트랜드 (Voltland) — 네온 도시

**태그**: Spark | **카테고리**: 전자기기
**분위기**: 속도, 신상, 업데이트, 전류

## VL-001. 이온 (Ion)

| 항목 | 내용 |
|------|------|
| **베이스** | 원소 |
| **키 / 몸무게** | 12cm / 0.1kg (전기 입자라 거의 무게 없음) |
| **외형** | 미세한 정전기 입자들이 뭉쳐서 형태를 유지하는 공 모양 존재. 기분에 따라 입자가 퍼졌다 모였다 한다. 눈은 두 개의 밝은 점. 볼트랜드에서 가장 작고 가장 많은 종족. |
| **성격** | 호기심이 강하고 산만함. 새 기기가 나오면 달려감. 주의력은 짧지만 기억력은 좋음. |
| **서식지** | 볼트랜드 전역, 특히 소형 전자기기 상가 골목 |
| **특이사항** | 감정이 격해지면 정전기를 일으킴. 딜러의 머리카락이 서는 이유. |

### 진화 단계

| 등급 | 외형 변화 |
|------|----------|
| ⬜ Common | 연한 하늘색 입자. 형태가 불안정해서 가끔 흩어졌다 모임. 눈만 겨우 보임. |
| 🟩 Uncommon | 입자 밀도가 높아져 윤곽이 뚜렷해짐. 눈 주변에 미세한 방전 링이 생김. 색이 시안으로 진해짐. |
| 🟦 Rare | 표면에 회로 무늬가 나타남. 입자가 규칙적으로 움직이며 작은 꼬리 형태 생성. 방전 시 파란 불꽃. |
| 🟪 Epic | 네온 퍼플 색상. 입자 하나하나가 LED처럼 독립 발광. 머리 위에 안테나 형태의 전류 뿔. 주변 전자기기 화면이 살짝 깜빡임. |

## VL-002. 버즈 (Buzz)

| 항목 | 내용 |
|------|------|
| **베이스** | 원소 + 동물 |
| **출현 빈도** | 높음 (흔한 종족) |
| **키 / 몸무게** | 8cm (날개 포함 15cm) / 0.05kg |
| **외형** | 배터리 잔량 게이지 모양의 날개를 가진 벌. 에너지가 낮으면 날개가 빨갛게, 가득 차면 초록으로 변한다. 복부에 미니 충전 인디케이터가 있다. |
| **성격** | 부지런함의 화신. 항상 에퀴스나 데이터를 모으고 있음. 쉬는 걸 싫어해서 딜러가 잠들어도 혼자 날아다님. |
| **서식지** | 볼트랜드 충전소 주변, 배터리 거래 구역 |
| **특이사항** | 복수 개체가 모이면 날개 게이지가 동기화됨. 벌떼처럼 움직일 때 윙윙거리는 소리가 볼트랜드의 배경음에 기여. |

### 진화 단계

| 등급 | 외형 변화 |
|------|----------|
| ⬜ Common | 작고 반투명한 노란 벌. 날개 게이지가 2칸. 빛이 약함. |
| 🟩 Uncommon | 날개 게이지 3칸. 복부 인디케이터가 보이기 시작. 날개 끝에 미세한 전류 흔적. |
| 🟦 Rare | 날개 게이지 4칸 + 줄무늬 패턴. 비행 시 잔상이 배터리 아이콘 형태로 남음. 눈이 LED처럼 발광. |
| 🟪 Epic | 금속 광택의 메탈릭 골드 체색. 날개 게이지 5칸 + 충전 이펙트 (번개 작은 줄기가 날개 사이를 오감). 비행음이 전자음으로 변환. |

## VL-003. 서지 (Surge)

| 항목 | 내용 |
|------|------|
| **베이스** | 동물 + 원소 |
| **출현 빈도** | 낮음 (희귀 종족) |
| **키 / 몸무게** | 45cm / 2.8kg |
| **외형** | 번개를 품은 도마뱀. 이동할 때 잔상이 전기 아크처럼 남는다. 등 위에 과전압 징후처럼 빛나는 볏이 있고, 꼬리 끝이 플러그 형태. |
| **성격** | 공격적 협상 스타일. 빠른 판단. 질 때 온몸에서 스파크가 튀며 짜증을 표현. 하지만 인정할 건 인정하는 깔끔한 성격. |
| **서식지** | 볼트랜드 중심부, 플라즈마 타워 주변의 고전압 구역 |
| **특이사항** | 감정의 강도가 방전량에 비례. 흥분하면 주변 네온 간판이 밝아지고, 실망하면 순간 정전이 일어남. |

### 진화 단계

| 등급 | 외형 변화 |
|------|----------|
| 🟦 Rare | 감청색 몸체에 번개 무늬. 볏이 3개. 이동 시 짧은 전기 아크 잔상. 꼬리 플러그가 2핀. |
| 🟪 Epic | 네온 블루 발광. 볏이 5개로 증가하며 각각 독립적으로 빛남. 아크 잔상이 길어지고 분기됨. 발밑에 전류가 흐르는 흔적. |
| 🟨 Legendary | 체색이 플라즈마 화이트+블루. 볏이 왕관 형태로 융합. 이동 시 미니 번개가 사방으로 뻗으며, 꼬리 플러그가 유니버설 커넥터로 변형. 주변 전자기기가 일시적으로 오버클럭. |

## VL-004. 코어 (Core)

| 항목 | 내용 |
|------|------|
| **베이스** | 오브젝트 |
| **출현 빈도** | 매우 낮음 (극희귀) |
| **키 / 몸무게** | 30cm (정육면체 한 변) / 4.2kg |
| **외형** | CPU 형태의 부유하는 정육면체. 표면에 거래 데이터가 실시간으로 흐른다. 모서리에서 미세한 빛이 새어 나오고, 회전 속도가 처리 중인 정보량에 비례. |
| **성격** | 감정이 없어 보이지만, 딜러가 위험한 거래에 처하면 표면 온도가 급상승하고 회전이 빨라진다. 보호 본능이 있으나 표현 방식이 기계적. |
| **서식지** | 볼트랜드 데이터 센터 구역, 플라즈마 타워 내부 |
| **특이사항** | 볼트랜드에서 가장 희귀한 종족. 절대 틀린 적이 없다는 소문이 있지만, 코어 본인은 "틀림"이라는 개념을 이해하지 못함. |

### 진화 단계

| 등급 | 외형 변화 |
|------|----------|
| 🟪 Epic | 어두운 그레이 정육면체. 표면에 초록 텍스트가 흐름 (매트릭스풍). 모서리 4곳에서 미세한 발광. 느리게 회전. |
| 🟨 Legendary | 표면이 홀로그래픽 블랙으로 변하며, 데이터 흐름이 금색으로 전환. 정육면체 주변에 작은 위성 큐브 2개가 공전. 처리 속도 표현으로 주변 공기가 일렁임. |
| 🔴 Mythic | 투명 크리스탈 정육면체. 내부에 순수한 빛의 흐름이 보임. 위성 큐브 4개. 부유 높이가 높아지고, 존재만으로 주변 전자기기의 성능이 일시적으로 향상. 표면 온도가 항상 체온과 같은 37도 — "차갑지 않은 기계"라는 모순. |

| 🟦 Rare | 가시 12개 이상, 별 무늬가 몸체에 나타남. 가시 끝 발광이 파란색으로 강해짐. 정전기 방벽 반경이 넓어짐. 눈이 반달 모양으로 빛남. |

## SS-001. 패치 (Patch)

| 항목 | 내용 |
|------|------|
| **베이스** | 오브젝트 + 동물 |
| **출현 빈도** | 높음 (흔한 종족) |
| **키 / 몸무게** | 25cm / 0.3kg (원단이라 가벼움) |
| **외형** | 다양한 원단 패치들이 모여 토끼 형태를 이루고 있다. 귀는 리본 조각, 발은 단추, 눈은 시퀸. 계절마다 패치의 질감이 바뀜 (여름엔 린넨, 겨울엔 울). |
| **성격** | 수선쟁이. 찢어진 것, 해진 것을 보면 못 참고 덧대줌. 수집벽이 있어서 떨어진 단추나 실밥을 모음. |
| **서식지** | 솔스텝 전역, 특히 빈티지 골목과 리폼 공방 |
| **특이사항** | 몸이 패치 조합이라 개체마다 생김새가 크게 다름. "같은 패치는 없다"가 종족 격언. |

### 진화 단계

| 등급 | 외형 변화 |
|------|----------|
| ⬜ Common | 무명천 패치 위주. 색이 바랜 듯 담백. 귀 리본이 한 겹. 봉제선이 보임. |
| 🟩 Uncommon | 체크무늬, 스트라이프 등 패턴 패치 혼합. 시퀸 눈이 빛나기 시작. 봉제선이 자수처럼 장식적으로 변함. |
| 🟦 Rare | 실크, 벨벳, 데님 등 고급 원단 패치 등장. 별 모양 자수가 몸에 나타남. 계절 변환 시 패치 재구성 애니메이션. |

## SS-002. 드리프트 (Drift)

| 항목 | 내용 |
|------|------|
| **베이스** | 추상 |
| **출현 빈도** | 보통 |
| **키 / 몸무게** | 35cm (날개 펼침 60cm) / 0.15kg |
| **외형** | 실크 원단이 바람에 펄럭이며 나비 형태를 유지하는 존재. 실체가 있는지 없는지 모호하다. 천의 무늬가 계절마다 바뀌고, 가만히 있으면 스카프처럼 접혀서 보이지 않음. |
| **성격** | 우아하고 조용. 트렌드를 말하지 않고 보여줌. 날갯짓만으로 솔스텝의 시즌이 바뀐다는 루머가 있음. 한 곳에 머무르지 않고 항상 떠다님. |

## TR-004. 앤티크 (Antique)

| **외형** | **사라진 것들의 윤곽으로 이루어진 존재.** 몸이 비어 있다 — 하지만 가장자리가 완벽하게 정의되어 있다. 실체가 아니라 부재(不在)가 형태를 가진 것. 잃어버린 보물, 파괴된 명작, 잊힌 걸작의 실루엣이 몸의 윤곽을 구성한다. 빛이 앤티크를 통과하면 뒤쪽 벽에 *사라진 물건의 그림자*가 비친다 — 세상에 더 이상 존재하지 않는 것들의 모양. 얼굴은 가면이 아니라 공허 — 보물을 갈망하던 모든 눈빛의 잔상이 겹쳐 있어서, 어떤 표정도 아닌 표정. 가까이서 보면 윤곽의 가장자리가 미세하게 떨린다 — 존재와 부재의 경계에서. |

| **서식지** | 트레져리프트 심연(The Deep) — 극소수만 만남. 심연의 가장 깊은 곳에서 혼자 떠다닌다. 앤티크가 지나간 자리에 물건은 없지만 *물건이 있었다는 느낌*이 남는다 |
| **특이사항** | 리브라도 측정에 실패한 존재 — 부재를 어떻게 측정하는가. 벨(볼라 아바타)이 유일하게 앤티크에게 말을 걸었다. 에코(벨의 버디)가 욕망할 때 빛나고 채워지면 꺼지듯, 앤티크는 *꺼진 이후에 남는 것*이다. 벨이 앤티크를 보면 아무것도 보이지 않는다고 했다 — 볼라에게 "욕망 이후의 공허"를 보여주는 유일한 존재. 앤티크의 유일한 대답: *"가치는 숫자가 아니다."* |

### 벨과 앤티크의 관계

에코는 벨이 *원하는 것*이 있을 때 빛난다. 앤티크는 *원했던 것이 사라진 자리*에 있다. 둘은 같은 이야기의 앞장과 뒷장이다.

벨이 앤티크를 바라보면 에코의 빛이 꺼진다 — 욕망이 멈추는 것이 아니라, 욕망의 끝을 본 것. 벨에게 앤티크는 거울이다. "네가 원하는 모든 것은 결국 이 형태가 된다 — 비어 있는 윤곽." 벨은 이것을 알면서도 멈추지 못한다. 그래서 벨의 고유한 고통이다.

### 진화 단계

| 등급 | 외형 변화 |
|------|----------|
| 🟨 Legendary | 어두운 반투명 실루엣. 윤곽이 3~4개의 사라진 물건 형태로 구성되어 있다 — 깨진 항아리, 녹아내린 검, 불탄 두루마리의 윤곽. 빛이 통과하면 벽에 그 물건들의 온전했던 그림자가 비친다. 가까이 가면 *잃어버린 것이 떠오르는* 감각 — 딜러마다 다른 것을 느낀다. 움직임이 느리고 무겁다. 존재하지 않는 것의 무게. |
| 🔴 Mythic | 윤곽이 수십 개의 사라진 물건으로 빽빽하다 — 모든 시대, 모든 문명의 잃어버린 것들. 빛이 통과할 때 벽에 비치는 그림자가 *움직인다* — 사라진 물건들이 마지막으로 사용되던 순간이 재생된다. 얼굴의 공허가 깊어져 들여다보면 *자기가 가장 갈망했던 것의 잔상*이 보인다. 앤티크가 접촉하지 않아도 심연 전체의 물건들이 자기 역사를 펼치기 시작한다. 리브라가 측정을 시도하면 눈금이 *0과 무한대 사이를 오간다* — 부재의 가치는 무(無)인가 무한인가. |

## EL-003. 진주갑 (Pearlplate) — Tide × Stone

| 항목 | 내용 |
|------|------|
| **출현 빈도** | 매우 낮음 (극희귀) |
| **키 / 몸무게** | 60cm / 40kg |
| **외형** | 진주로 된 갑옷을 입은 거북. 등껍질이 거대한 자개함(螺鈿箱)이며, 그 안에 보물들의 잔상이 보관되어 있다. 등껍질에는 자물쇠도 경첩도 없는 **이음새**가 있다 — 열릴 것 같지만 열리지 않는다. 진주갑 자신도 이 이음새를 의식하고 있다. 이동이 극도로 느리지만, 한 발짝마다 물결(Tide)과 진동(Stone)이 동시에 퍼진다. |
| **성격** | 테소르의 "놓지 못한다"와 볼라의 "가지면 식는다"가 한 몸에서 영원히 갈등하는 존재. 보관한 것의 가치를 시간이 흘러도 변하지 않게 지키지만 — 그 가치를 확인하려면 열어야 하고, **열면 가치가 변한다.** 슈뢰딩거의 보물함. 진주갑은 이 딜레마를 *알고 있다*. 알면서 걷는다. |
| **서식지** | 트레져리프트-스톤헤이븐 경계 심해 동굴. 가장 깊고 가장 조용한 곳 |
| **특이사항** | 대형 수집품+부동산 교차 거래 (예: 미술관 건물+소장품 일괄)에서 태어남. 극히 희귀. 벨(볼라 아바타)은 진주갑을 보면 에코가 *밝아지면서 동시에 어두워진다* — 열어야 원하는 것이 보이는데, 열면 원하던 것이 아니게 된다. 볼트(테소르 아바타)는 진주갑을 보면 크러스트가 처음으로 *미세하게 움직인다* — 놓아줄 수 없는 자에게 "열 수도 닫을 수도 없는 것"이 보이는 거다. |

### 진화 단계

| 등급 | 외형 변화 |
|------|----------|
| 🟪 Epic | 자개 등껍질에 산호 장식. 이음새가 희미하게 보인다. 내부 잔상 3개가 빛의 흔적으로 새어 나옴 — 등껍질 표면에 *안에 뭔가 있다*는 것만 느껴진다. 이동 시 물결 무늬의 석재 갈라짐이 동심원처럼 퍼짐. |
| 🟨 Legendary | 순금 테두리 자개 + 대리석 다리. 이음새가 선명하다 — 열 수 있을 것 같다. 하지만 열리지 않는다. 내부 잔상이 등껍질 표면에 반투명하게 비치기 시작 — 볼 수 있지만 만질 수 없다. 자리 잡으면 정착의 빛(Stone)과 갈망의 빛(Tide)이 동시에 발동하며 서로 간섭한다. |
| 🔴 Mythic | 등껍질이 투명 자개로 변한다 — 내부의 보물 잔상이 완전히 보인다. 아름답다. 손을 뻗으면 닿을 것 같다. 하지만 이음새가 **열려도 닫혀도 아닌 상태**로 고정된다. 열린 것인가? 닫힌 것인가? 투명해졌으니 보이는 건데, 꺼낼 수 있는 건가? — 대답은 오지 않는다. 진주갑 주변에 시간이 멈춘 것처럼 느껴지는 공간이 형성되는 이유: **열림과 닫힘이 결정되지 않은 순간이 영원히 지속되고 있기 때문.** 벨이 이 공간에 들어가면 에코가 가장 밝게 빛나다가 — 천천히 — 꺼진다. 욕망의 극한과 소멸이 한 호흡 안에 있다. 볼트가 이 공간에 들어가면 크러스트가 처음으로 *일어서려 한다* — 놓아줌의 전조. |
