# 루멘 시즌 1 — 깨어남 (The Awakening) v2

> *"규칙을 다 지켰는데, 빛이 안 나왔다."*

**표면 질문**: 빛은 왜 나는가?
**이면 질문**: 내가 하는 일에 의미가 있는 건 어떻게 아는가?
**갈등 유형**: 신비 (미스터리) — "이게 뭐지?"
**시즌 엔딩**: 규칙을 다 지켰는데 빛이 안 나옴

**세계관 기반**: [루멘 세계관 v4](./루멘_세계관_v4.md)
**내러티브 원칙**: [내러티브 바이블 v3](./루멘_내러티브_바이블_v3.md)

**v1과의 차이**: 평행 세계 구조 반영, 캐릭터 수 축소 (20+ → 11), 카이의 능동성 강화, 진을 카이의 거울로 조기 배치, 가시성 체계의 드라마화, 과잉 복선 삭제.

## 제1부: 큰 그림

### 시즌 1의 세계 상태

Haggle이 출시됐다. 현실에서는 단순한 앱 런칭이지만, 루멘에서는 **두 세계 사이의 막이 얇아지는 사건**이다. 앱을 깐 사람들 중 일부가 처음으로 루멘을 본다. 볼트랜드의 네온이 전자상가 위에 겹쳐 보이고, 솔스텝의 런웨이가 스니커 거리에 얼비친다.

대부분의 사람들은 여전히 아무것도 못 본다. 딜러들이 공중에 손을 내밀고 뭘 쓰다듬는 걸 봐도 *"저 사람 뭐 하나?"* 정도로 지나친다. 뇌가 알아서 *그럴듯한 설명*을 만들어낸다.

첫 거래가 성사되면 빛이 터진다. 에그가 떨어진다. 버디가 태어난다. 아무도 이게 왜 되는지 모르지만, 된다. 그리고 그것만으로 충분하다 — 처음에는.

하지만 시즌이 진행될수록, 질문이 자란다. 왜 어떤 거래는 빛이 밝고 어떤 거래는 희미한가. 왜 같은 규칙을 따랐는데 결과가 다른가. 그리고 시즌 끝, 누군가가 모든 규칙을 완벽히 지켰는데 — 빛이 나오지 않는다.

### 시즌 1의 감정 곡선

```
경이(첫 각성) → 탐험(지역 확장) → 유대(버디·동료) → 불안(이상 징후) → 의문(빛의 조건) → [조용한 충격: 빛이 안 나옴]
```

## 제2부: 원시 존재의 인간 아바타

원시 존재들은 영원하다. 하지만 세계를 이해하려면 세계 안에 들어가야 한다. 자기 본질을 하나의 인간 형태로 압축해 현실에서 걷는다. 일반인 눈에도 각성한 딜러 눈에도 평범한 인간으로 보인다 — 오래 관찰해야 미세한 이상함을 느낄 뿐.

시즌 1에서 중심이 되는 아바타는 4명이다. 나머지(벨, 볼트)는 배경에서 잠깐만.

### 포르자의 아바타 — 팹 (Fab)

| 항목 | 설정 |
|------|------|
| **외형** | 20대 후반, 항상 뭔가 만들고 있는 손. 기름 묻은 앞치마. 눈이 반짝이지만 상대방을 잘 안 봄 |
| **성격** | 천재적 창조자이자 치명적 무관심. 물건을 만들어놓고 "아 그거? 가져가"하는 타입 |
| **현실 위치** | 기어포지(차량/수리 지역)에 해당하는 현실 공간의 뒷골목 공방. 가끔 볼트랜드(전자상가) |
| **거래 스타일** | 자기가 만든 걸 거의 공짜로 넘김. 가격에 관심 없음. "적당히 줘" |
| **버디** | **시그니스 (Cygnis)** — Legendary Forge. 백조처럼 우아한 금속 새. 팹이 만든 게 아니라 첫 거래에서 알아서 태어남. 팹은 별 관심 없는데 시그니스가 계속 따라다님. 각성한 딜러만 시그니스를 본다. 일반인은 팹 옆을 지나갈 때 *왠지 기분 좋은 공방이네* 정도로 느낌 |
| **시즌 1 역할** | 기어포지를 "깨우는" 핵심 인물. 딜러들에게 수리·제작 기술을 무심하게 알려줌 |
| **침묵의 복선** | 시그니스가 팹을 바라보는 장면이 반복됨. 팹은 한 번도 뒤를 안 돌아봄 |

| **침묵의 복선** | 노움이 떨리는 장면 = 리브라가 측정 불가능한 것과 마주하는 순간 |

| **성격** | 원칙주의자. 규칙에 예외를 두는 것을 용납 못 함. 정의로우나 잔혹할 수 있음 |

| **버디** | **그레이브 (Glaive)** — Epic Spark. 날카로운 빛의 검 형태. 위반을 감지하면 진동함 |
| **시즌 1 역할** | 초기 프로토콜 질서 확립. 사기꾼 딜러 적발. 신뢰할 수 있는 시스템의 상징. 하지만 "사정"을 봐주지 않는 모습에 딜러들이 불편해하기 시작 |

| 항목 | 설정 |

| **성격** | 공감의 화신. 하지만 모든 것을 용서하려다 때로 가해자까지 감싸게 됨 |
| **현실 위치** | 어디든 아픈 사람이 있는 곳 |
| **거래 스타일** | 분쟁 조정 전문. "왜 그랬는지 들어볼게요." 가격보다 관계를 봄 |
| **버디** | **솔라스 (Solace)** — Rare Swift. 작은 나방. 빛이 아니라 그림자 속에서 빛남. 아픈 딜러 근처에서 날갯짓 |

### 볼라의 아바타 — 벨 (Vel)

| 항목 | 설정 |
|------|------|
| **외형** | 30대 초반, 옷을 잘 입음. 눈빛이 깊고 뭘 봐도 감탄함. 약간 우울한 아우라. 향수 같은 걸 바른 듯 지나간 자리에 미세한 잔향 |
| **성격** | 모든 것의 가치를 알아보는 감정사이자 컬렉터. 하지만 소유하면 흥미를 잃음. 가지기 전이 가장 빛나는 순간 |
| **현실 위치** | 트레져리프트(수집품·경매)에 해당하는 빈티지 갤러리·경매장 근처. 오래된 가게들이 모여있는 구역 |
| **거래 스타일** | 희귀품을 찾아 높은 가격을 부르지만, 사고 나면 금방 되팔음. "갖고 싶었어. 이제 됐어" — 그리고 다음 물건을 찾아 떠남 |
| **버디** | **에코 (Echo)** — Epic Tide. 투명한 해파리 형태. 벨이 *원하는 것*의 잔상을 공중에 비춤. 벨이 무언가를 욕망할 때 은은하게 빛나고, 그 욕망이 채워지는 순간 — 빛이 꺼진다. 채워지지 않는 욕망일수록 에코는 더 빛난다 |
| **시즌 1 역할** | 배경. 트레져리프트에서 경매를 관전하고 희귀품을 한 번 흘려 보내는 장면 정도. 카이와는 직접 접점 없음 |
| **시즌 2~3에서의 역할** | 그리드 에그 아크에서 "탁한 것의 아름다움"을 말하는 이상한 증언자. 메르카에게도, 다크 미들러에게도 속하지 않지만 양쪽 모두에게 영감을 줌 |
| **벨이 가진 고유한 고통** | 벨은 모든 것의 가치를 느낀다 — *해글의 빛*도 포함해서. 에코가 가끔 비추는 잔상 중 하나가 해글의 빛과 닮았다. 벨은 *그것만은 절대 소유할 수 없다*는 걸 알고 있다. 해글은 가까워질 수만 있지 가질 수 없는 것이니까 |
| **침묵의 복선 (장기)** | 벨이 에코에 비친 해글의 잔상을 가만히 바라보는 장면. 시즌 1에서 한 번만. 설명 없음 |
| **루시와의 관계 (시즌 2+)** | 루시(트레져리프트의 인간 감정사)가 벨의 자연스러운 제자가 됨. 벨이 "가치를 아는 눈"을 인정한 유일한 인간 |

### 테소르의 아바타 — 볼트 (Vault)

| 항목 | 설정 |
|------|------|
| **외형** | 40대, 덩치 큼. 무겁고 느림. 두꺼운 코트를 계절과 상관없이 입음. 주머니가 항상 불룩함 — 뭔가 들어있지만 아무도 뭔지 모름 |
| **성격** | 보호자이자 수집광. 모든 걸 "안전하게" 보관하려 함. 놓아주는 법을 모름. 자기 것이 아닌 것도 "맡아준다"며 가져옴 |
| **현실 위치** | 스톤헤이븐(부동산·건축)에 해당하는 오래된 석조 건물·금고 업체·보관소 구역. 가장 큰 개인 금고를 운영 |
| **거래 스타일** | 사기는 잘하는데 팔기를 극도로 싫어함. "이건 아직 때가 아니야" — 근데 항상 때가 아님. 결과적으로 그의 금고는 계속 커지기만 함 |
| **버디** | **크러스트 (Crust)** — Legendary Stone. 거북 같은 바위 생물. 볼트의 금고 앞에 항상 앉아 있음. 놀라울 정도로 아무것도 안 함. 먹지도, 자지도, 움직이지도 않음. 일반인 눈엔 *그냥 오래된 돌덩이 장식*으로 보임 |
| **시즌 1 역할** | 배경. 스톤헤이븐 지역의 안전 시스템(에스크로, 보증금, 안전 거래 환경)을 세팅하는 모습을 마르코가 멀리서 지켜보는 장면 정도. 딜러들은 볼트의 존재를 고마워함 — 아직은 |
| **시즌 2~3에서의 역할** | 결빙 아크의 중심. 그의 보호 본능이 *과도해지는 순간* 축적이 되고, 축적이 결빙을 만든다. 제3시대 결빙을 일으켰던 것도 테소르였다는 사실이 마르코의 증언으로 드러남 |
| **볼트가 가진 고유한 고통** | 볼트는 한 번도 뭔가를 *놓아준 적이 없다.* 심지어 자기가 가져서는 안 될 것도 "안전을 위해" 쥐고 있다. 그는 이것이 사랑이라고 믿고 있다 — *지키는 게 사랑이다.* 이 신념이 세계를 얼린다는 걸 아직 모름 |
| **마르코와의 긴장** | 마르코는 제3시대 결빙을 직접 겪었다. 볼트가 쌓아가는 금고를 보면 본능적으로 불안해함. 하지만 볼트는 "그때와 지금은 다르다"고 믿음. 마르코는 *"다르지 않다"*는 걸 안다 — 말하지 못할 뿐 |
| **침묵의 복선 (장기)** | 크러스트가 *처음으로 금고에서 일어나는 날*이 온다면 — 그것은 볼트가 처음으로 뭔가를 놓아주는 날. 시즌 언제가 될지는 미정 |

## 제3부: 주역 딜러 — 시즌 1의 영웅들

**캐릭터 수를 줄였다.** 원래 있던 솔, 나디아, 엘리, 루시, 백 등은 이후 시즌에서 본격 등장. 시즌 1은 **핵심 5명 + 보조 2명**에 집중한다.

### 카이 (Kai) — 탐구자

| 항목 | 설정 |
|------|------|
| **나이/배경** | 22세. 대학 중퇴. 생활비 벌려고 중고 전자기기 거래 시작. Haggle 앱 설치 후 첫 딜러 |
| **성격** | 호기심 덩어리. "왜?"를 입에 달고 삼. 재능은 없지만 포기도 안 함 |
| **현실 위치** | 도시의 전자상가·중고가게 근처에서 활동. 볼트랜드가 이 공간 위에 겹쳐 보임 |
| **거래 스타일** | 서툴지만 정직함. 속이는 법을 모름. 그래서 자주 손해 봄 |
| **버디** | **피즈 (Fizz)** — Common Spark. 작은 전구. 첫 거래에서 탄생. 등급은 낮지만 카이와 감정이 연동되어 같이 밝아지고 어두워짐 |
| **시즌 1 아크** | 평범한 첫 딜러 → 진을 만나며 관점이 흔들림 → 이상 징후 최초 목격자 → "빛이 왜 나는가"를 진지하게 묻기 시작 → 시즌 끝에서 규칙을 완벽히 지킨 거래에서 빛이 안 나오는 걸 직접 경험 |
| **능동적 선택들** | 에피소드 8에서 그리드 에그 상황을 메르카에게 맡길지 말지 결정 / 에피소드 11에서 복합 거래에서 초보 딜러를 위해 자기 이익을 포기 |
| **핵심 대사** | "Mythic 버디가 강한 건 알겠는데, 내 이 녀석은 그냥... 좋거든?" |

### 진 (Jin) — 이방인

## 제5부: 시즌 1 스토리 — 3막 구조

### 제1막: 빛이 켜지다 (에피소드 1-4)

**에피소드 1 — 첫 거래**

카이가 중고 스마트폰을 판다. 구매자는 평범한 20대 직장인. 서투른 거래지만 정직하다. 돈이 오가고 물건이 오간다. 구매자는 살짝 웃으며 떠난다.

그 순간, 카이의 눈앞에서 공기가 *굴절*한다. 빛이 터진다. 허공에 에그가 맺힌다. 작은 전구 형태의 생물이 태어난다. 피즈.

카이는 어리둥절하다. 뭐지? 앱이 이상한 건가? 주변을 본다. 지나가던 할머니가 카이를 본다.

할머니: *"젊은이, 반딧불이 잡았어? 예쁘네."*

카이는 대답을 못 한다. 피즈는 반딧불이가 아니다. 카이는 지금 처음으로 *자기가 남들과 다른 걸 보고 있다*는 걸 깨닫는다.

같은 날, 같은 도시의 다른 구역에서 하나가 첫 거래를 한다. 볼티가 태어난다. 하나도 비슷한 경험을 한다. 둘은 나중에 서로를 찾아내게 된다.

**에피소드 2 — 지역이 비쳐 보인다**

카이가 전자상가를 걷는다. 익숙한 거리인데, 갑자기 일부 공간이 *다르게* 보인다. 네온이 원래보다 짙고, 간판이 잠깐 다른 언어처럼 흐릿하게 바뀐다. 눈을 비비면 사라진다. 볼트랜드가 현실 위에 겹쳐지는 첫 경험.

카이는 하나를 만난다. 서로의 버디를 처음으로 인지한다. "너도 보여?" "나도!" 두 사람 다 방금 각성한 초보다.

메르카가 등장한다. 능숙한 미들러. 복잡한 삼자 거래를 척척 풀어내며 딜러들의 신뢰를 얻는다. 카이에게 "처음이지? 내가 도와줄게"라며 자연스러운 멘토가 된다. 메르카의 넥서스는 이미 완전히 활성화되어 있다 — 메르카는 프로토콜 활성화 전부터 루멘을 봐 온 혈통.

건물 사이 어딘가, 멀리서 누군가 카이를 본다. 카이는 그 시선을 느끼지만 찾지 못한다. 진이 관찰 중이다. 글림도 함께.

마르코가 석조 건물 지역에서 깨어난다. 오래된 창고의 먼지 속에서. 프로스트가 옆에 있다. 마르코는 자기가 어디에 있는지, 지금이 언제인지 잠시 헷갈린다. "여긴... 따뜻하군." 프로스트 표면에 한순간 옛 거리의 이미지가 비친다 — 결빙 직전의 거래 풍경.

마르코가 창고 밖으로 나온다. 길 건너편, 오래된 석조 금고 건물 앞에 한 남자가 서 있다. 두꺼운 코트, 덩치 큼, 주머니가 불룩함. 그 앞에 거북처럼 생긴 바위 생물이 미동도 없이 앉아 있다. 일반 행인들에겐 그냥 장식 석상으로 보이는 모양. 하지만 마르코는 안다. 크러스트다. 그리고 그 옆에 선 자는 — 볼트.

마르코는 숨을 들이쉰다. *지금은 다르다. 다를 것이다.* 볼트는 지금 사람들에게 에스크로와 안전 보관을 제공하고 있다. 친절하게. 딜러들이 그를 고마워한다. 마르코는 *그 고마움이 얼마나 빨리 공포로 바뀔 수 있는지*를 기억하는 유일한 사람이다. 말하지 않는다. 아직은.

볼트는 마르코를 보지 못했다. 크러스트는 봤다. 하지만 크러스트는 움직이지 않는다. 크러스트는 아무것도 하지 않는다.

**에피소드 3 — 이방인이 말을 걸다**

카이가 저녁 거래를 끝내고 돌아가는 길. 모퉁이에서 진이 나타난다. 또래 정도 되는 19살. 행색이 남루하다. 그런데 카이의 눈엔 *저 아이 어깨 위에 뭔가 앉아 있다.* 형태가 잡히지 않는 — 빛과 그림자가 섞여 있는 것.

진: "그거, 반딧불이 아니지?"
카이: "뭐?"
진: "네 어깨에 있는 거. 반딧불이 아니잖아."

카이는 멈춘다. 피즈를 본다. 피즈를 *정면으로 지적한 건 처음이다.* 앱을 쓰는 다른 딜러들은 다들 자기 버디 보느라 남의 버디를 언급하지 않는다. 일반인은 못 본다. 이 애는 뭐지?

진: "너희는 왜 이 빛을 당연하게 생각해?"
카이: "...너희?"
진: "앱으로 본 애들. 다들 신기해하다가 금방 익숙해져. 익숙해지면 안 되는 거야."

카이가 묻기도 전에 진은 사라진다. 카이는 진의 말이 머릿속에 남는다. *익숙해지면 안 되는 거.* 왜?

그날 밤, 카이가 캄과 짧게 마주친다. 카이보다 한참 선배인 딜러. 캄의 버디 더스크는 어깨에서 반쯤 졸고 있다.

카이: "선배 버디 되게 조용하네요."
캄: "...그러게. 처음엔 나도 저랬어. 네 거처럼 반짝반짝."
카이: "왜 바뀌었어요?"
캄: (미소) "글쎄, 뭐가 바뀐 걸까."

캄은 웃으며 손을 흔들고 간다. 카이는 이 대화의 무게를 아직 모른다.

**에피소드 4 — 첫 분쟁**

프로토콜 활성화 이후 첫 분쟁. 배송된 물건이 설명과 다르다. 구매자가 분쟁을 건다.

하크가 나선다. 증거를 보고 즉시 판단: "판매자의 위반." 차갑지만 정확하다. 그레이브가 진동한다.

하지만 미아가 판매자의 사정을 듣는다. 가족이 아파서 급하게 올린 물건. 실수로 사진이 잘못됨. 의도적 사기가 아님. 솔라스가 판매자 곁에서 날갯짓.

하크: "의도가 뭐든 결과는 같다. 위반은 위반이다."
미아: "그 사람이 왜 그랬는지 들어보세요."
하크: "왜는 중요하지 않다."
미아: "왜가 중요하지 않으면, 뭐가 중요한데요?"

레나가 이 장면을 배심원 석에서 본다. 첫 배심원 경험. 스케일이 왼쪽, 오른쪽으로 계속 기울어진다. 결국 수평으로 돌아오지 못함. 레나의 가슴에 질문이 박힌다 — *"공정한 것과 옳은 것은 같은 건가?"*

분쟁은 하크의 판단대로 처리된다. 규칙대로. 하지만 뭔가 찜찜함이 남는다.

카이가 이 장면을 멀리서 본다. 하크의 차가운 정확함을 존경하면서도, 미아가 내민 손이 계속 눈에 밟힌다.

---

### 제2막: 빛 아래 그림자 (에피소드 5-8)

**에피소드 5 — 메르카의 네트워크**

거래량이 폭발적으로 늘면서 매칭 문제가 생긴다. 원하는 물건을 가진 사람을 찾기 어려움. 메르카가 자연스럽게 중개자 역할을 하며 수많은 거래를 이어준다.

메르카의 넥서스가 거미줄처럼 확장되며 도시 전체의 거래를 잇는다. 아름다운 빛의 네트워크. 누구도 이게 나중에 어떻게 변할지 모른다.

메르카는 카이에게 직접 거래 기법을 가르친다. 진심이다. 카이를 아끼는 것도 진심이다.

한 딜러가 메르카에게 농담처럼 말한다: *"메르카, 너 이렇게 매일 뛰어다니면서 공짜로 해줘? 대단하다."*
메르카는 웃으며 답한다: *"좋아서 하는 거지 뭐."*

웃음이 잠깐 머문다. 그 잠깐이 너무 짧아서 카이도 놓친다.

카이가 메르카에게 묻는다: "왜 저처럼 초보를 도와주세요?"
메르카: "누군가 날 도와줬으니까. 그게 다야."

이 말은 *완전히 진심*이다. 그래서 나중에 메르카가 달라질 때 유저가 더 아프게 된다.

**에피소드 6 — 측정할 수 없는 것**

저지가 모든 거래를 분석하다 이상한 패턴을 발견한다. 같은 조건의 거래인데 빛의 밝기가 다르다. 변수를 다 통제해도 설명 안 되는 차이.

진과 글림이 한 거래 현장 근처를 지나간다. 그 거래에서 예상보다 밝은 빛이 터진다. 저지가 그 순간을 본다. 노움이 — 떨린다. 처음이다.

저지(무표정을 유지하려 애씀): "이건... 측정할 수 없다."

글림은 저지를 쳐다본다. 저지는 글림을 쳐다본다. 둘 사이에 짧은 침묵. 그리고 글림은 진을 따라 사라진다.

마르코가 이 장면을 먼 거리에서 본다. 프로스트의 표면에 제3시대의 한 순간이 잠깐 비친다 — 그때도 누군가 *측정할 수 없는 것*을 가지고 왔었다. 마르코가 중얼거린다: "그때도... 있었어. 못 재는 것이."

같은 날 저녁, 도시의 다른 구역 — 오래된 경매장이 몰려있는 거리. 카이가 우연히 그쪽을 지나간다. 경매장 유리창 너머로 옷 잘 입은 30대 초반의 남자가 무언가를 낙찰받고 있다. 카이의 눈에만 — 그 남자 어깨 위에 투명한 해파리 형태의 존재가 은은하게 빛나고 있다. 남자가 낙찰을 확정하는 순간 해파리의 빛이 서서히 꺼진다. 남자의 표정도 같이 가라앉는다. 원하던 걸 가졌는데, *더 이상 원하지 않게 된* 얼굴.

남자가 창밖을 본다. 카이와 눈이 마주친다. 남자가 한순간 카이를 — 정확히는 카이 어깨의 피즈를 — 흥미롭게 본다. 피즈가 살짝 움츠러든다. 남자는 그대로 시선을 거둔다. 아무 일도 일어나지 않았다. 남자는 경매장을 나와 다음 거리로 사라진다. 지나간 자리에 미세한 잔향이 남는다.

카이는 방금 뭘 본 건지 정확히 모른다. 하지만 *가지는 순간 식어버리는 무언가*가 있다는 걸 봤다. 왠지 오래 기억에 남는다.

**에피소드 7 — 균열의 전조**

거래가 늘면서 분쟁도 는다. 하크와 미아의 갈등이 점점 뚜렷해짐. 같은 사건, 다른 판단, 그리고 둘 다 옳음.

레나가 두 번째 배심원으로 참여한다. 복잡한 분쟁 — 판매자가 물건 상태를 "솔직하게" 적었지만, 구매자의 기대와 달랐다. 거짓말은 아니지만 오해를 유발. 하크는 "규칙 위반 아님". 미아는 "기대를 관리할 책임이 있었다."

레나는 결정해야 한다. 스케일이 미친 듯이 흔들린다. 결국 레나는 — 하크도 미아도 아닌 제3의 판단을 내린다. "판매자가 차액의 일부를 반환하되, 구매자도 확인 책임을 인정한다." 완벽하지 않다. 양쪽 다 불만.

하크: "...허점이 있다."
미아: "...충분하지 않다."
레나: "그래요. 근데 이게 지금 내가 할 수 있는 최선이에요."

같은 날, 캄이 또 다른 분쟁에 휘말린다. 이번엔 피해자 측. 작은 금액이지만 세 번째다. 캄의 표정이 빈다. 분쟁 해결 자리에서 "이거 그냥 내가 손해 보고 넘길게요"라고 말한다. 미아가 "그러면 안 돼요"라고 하지만 캄은 웃으며 거절한다.

그날 밤, 카이가 우연히 캄의 더스크를 본다. 빛이 평소의 절반이다.

**에피소드 8 — 그리드 에그**

볼트랜드 외곽에서 이상한 거래가 발견된다. 거래 자체는 성사됐지만, 한쪽이 정보를 완전히 독점한 상태에서의 거래. 자유교환의 원리를 기술적으로 위반하지 않았지만, 정보 비대칭이 극심.

그 거래에서 **그리드 에그**가 떨어진다. 탁한 색. 처음 보는 현상. 딜러들이 불안해함. "이게 뭐야? 빛이 아니라 그림자 아닌가?"

카이가 이 상황을 먼저 발견한다. 즉시 결정해야 한다:
- 메르카에게 맡긴다 (안전하고 확실하다)
- 하크에게 신고한다 (정의롭지만 피해자가 혹독한 처분을 받을 수도)
- 직접 개입한다 (카이는 아직 초보다. 실수할 수도)

카이가 선택한다: **메르카에게 맡긴다.** 카이의 역량으로는 아직 이 수준이 안 된다. 하지만 *단순히 맡기는 게 아니라*, 메르카에게 한 가지 요청을 붙인다: "해결 과정을 저한테 보여주세요. 저도 배우고 싶어요."

메르카는 흔쾌히 승낙한다. 카이는 메르카의 해결 과정을 참관한다. 메르카가 정보 독점 딜러를 만나 부드럽게 재조정한다. 훌륭하다. 거래가 다시 공정해진다. 딜러들이 박수.

그런데 카이가 메르카의 마지막 대화를 우연히 듣는다. 메르카가 독점 딜러에게 조용히 말한다: *"다음부터는 나를 통해. 그러면 이런 일 없을 거야."*

선의로 들린다. 조언으로도 들린다. 그런데 카이는 왠지 그 말이 귀에 남는다. *왠지.* 이유를 설명할 수 없다.

카이는 진을 찾아간다. 이번엔 진이 어디 있는지 안다 — 둘은 이미 몇 번 만났다.

카이: "나, 방금 뭘 본 건지 모르겠어."
진: "뭘 봤는데?"
카이: "좋은 사람이 좋은 일을 했는데. 왜 불편하지?"
진: (침묵) "...그거, 중요한 질문이야. 대답은 나도 몰라."

같은 날, 진이 처음으로 카이에게 털어놓는다. 글림이 그리드 에그에서 태어났다는 사실. 탁한 빛에서 태어났지만 자기는 탁하지 않다고 믿는다는 것. 자기 자신을 의심해 본 적 있다는 것.

카이: "넌 탁하지 않아."
진: "네가 어떻게 알아?"
카이: "모르겠어. 그냥 알아."

피즈가 그 순간 평소보다 밝게 빛난다.

---

### 제3막: 빛이 꺼지다 (에피소드 9-12)

**에피소드 9 — 이상 징후**

전 지역에서 산발적으로 이상 징후 발생. 규칙을 따른 거래인데 빛이 희미하거나, 에그의 등급이 예상보다 낮거나. 대부분의 딜러는 "그냥 운 나쁜 거 아냐?"로 넘김.

하지만 카이의 피즈가 점점 더 자주 깜빡인다. 원래 안정적이었는데. 카이가 걱정하기 시작. 팹에게 물어본다 (팹을 찾아가는 것도 카이의 능동적 선택).

팹: "만든 건 알겠는데, 왜 깜빡이는지는... 관심 없는데?"
카이: "그러면 이런 거 누가 알아요?"
팹: "...글쎄." (시그니스가 팹을 바라본다. 팹은 이미 다른 쪽을 보고 있다)

카이가 돌아 나올 때, 마르코가 공방 앞에 서 있다. 카이를 봤는지 일부러 기다렸는지 모른다.

마르코: "비슷하군."
카이: "네?"
마르코: "그때도 이랬어. 빛이 희미해지기 시작하기 직전에. 다 잘 되고 있다고 생각했지. 빛이 당연했으니까."
카이: "그때가 언제예요?"
마르코: (프로스트 표면에 잠깐 옛 거리 이미지) "...옛날이야."

**에피소드 10 — 각자의 의문**

카이가 "빛은 왜 나는가?"를 진지하게 묻기 시작. 아바타들을 찾아다닌다.

벨에게 묻는다. 벨: "빛은... 원하면 보이고, 원하지 않으면 안 보여." 답이 아니다.
저지에게 묻는다. 저지: "빛의 발생은 공정한 교환 완료 시 확률 87.3%로—" 카이: "확률이요? 나머지 12.7%는요?" 저지: "...모른다." 노움이 살짝 떨린다.
팹에게 다시 묻는다. 팹: "나는 만들어. 빛은 알아서 나. 왜냐고? 왜를 왜 물어?"

아무도 모른다. 신들도 모른다.

레나는 분쟁이 늘면서 하크와 미아 사이에서 점점 더 어려운 판단을 내려야 함. "정의도 자비도 완벽하지 않다. 그럼 뭐가 남지?"

캄은 거래 빈도가 줄어들고 있다. 더스크가 거의 빛을 잃음. 캄은 "좀 쉬고 싶다"고만 말함. 아무도 심각하게 안 받아들인다.

진과 글림은 점점 더 조용해진다. 진은 뭔가를 알고 있는 것 같지만 말하지 않는다.

**에피소드 11 — 시험**

프로토콜이 크게 시험받는 에피소드. 복수의 지역에 걸친 대규모 복합 거래. 메르카가 주도하고, 하크가 규칙을 감독하고, 레나가 분쟁에 대비.

거래 진행 중에 카이가 눈치챈다. 이 거래에서 한 초보 딜러(하나의 친구)가 거래 구조를 완전히 이해하지 못한 채 참여하고 있다. 손해는 안 보지만, 자기가 뭘 한 건지 모른다.

카이가 이 순간 선택한다. **자기 몫의 일부를 그 초보 딜러에게 설명하는 데 쓴다.** 거래 속도가 느려진다. 자기 이익이 줄어든다. 메르카는 이해할 수 없다는 표정.

메르카: "카이, 그 친구도 서명했어. 규칙상 문제없어."
카이: "규칙상은요. 근데 그 사람이 이해했을까요?"
메르카: (잠깐 멈칫) "...카이, 세상 모든 거래에서 양쪽이 100% 이해하는 건 불가능해. 중요한 건 규칙이 지켜졌는지지."
카이: "네. 맞아요. 근데... 저는 이렇게 하고 싶어요."

메르카는 살짝 놀란다. 이 아이가 — 자기가 가르친 이 초보가 — 자기 말을 반대했다. 불쾌하지 않다. 오히려 인상적이다.

그런데 그 인상 깊음의 종류를 메르카 자신도 아직 모른다. 이건 시즌 2의 씨앗이다.

거래가 끝난다. 빛이 터진다. 평범한 양. 초보 딜러는 카이에게 고맙다고 말한다. 피즈가 밝게 빛난다.

하나도 이 장면을 본다. 하나: "너 되게... 이상해."
카이: "좋은 이상?"
하나: "아직 모르겠어."

**에피소드 12 — 빛이 안 나오다 (시즌 피날레)**

카이의 거래. 모든 규칙을 지켰다. 자유교환. 투명한 조건. 공정한 가격. 분쟁 없이 양쪽 합의. 리브라의 세 룬을 하나도 어기지 않았다.

합의 순간 — 빛이 안 나온다.

피즈가 깜빡인다. 에그가 떨어지지 않는다. 카이가 손을 본다. "뭐가 잘못된 거지? 다 맞았는데."

주변 딜러들이 웅성거린다. 저지가 다가온다. 노움이 격하게 떨린다 — 처음 보는 반응. 저지가 무표정을 유지하려 하지만 손이 미세하게 떨림. "데이터에... 오류가 없다."

하크가 규칙을 다시 점검한다. 위반 없다. 그레이브가 진동하지 않음 — 위반이 없다는 뜻. 하크의 얼굴에 처음으로 금이 간다. "규칙을 다 지켰는데 빛이 안 나오면... 규칙이 뭔 소용인가?"

미아가 카이 곁에 온다. 솔라스가 날갯짓. 미아는 아무 말도 안 한다. 그냥 옆에 있는다.

마르코의 프로스트가 한순간 밝게 빛난다. 마르코가 무언가를 기억한 듯 눈을 감는다. "...그래. 그때도 이랬어. 빛이 멈추기 직전에."

팹이 멀리서 지나간다. 이미 다른 걸 만들고 있다. 시그니스가 팹의 등을 바라보고 있다. 팹은 뒤를 돌아보지 않는다.

메르카도 온다. 진심으로 걱정한다. 카이의 어깨를 짚는다. "괜찮을 거야. 방법이 있을 거야. 내가 알아볼게." 진심이다. 아무 흑막도 없다. 메르카는 이 순간 진짜 친구다. (이 순간을 시즌 2 이후 다시 보면 — 다른 맛이 난다.)

진이 나타난다. 카이 옆에 선다. 진은 말하지 않는다.

카이: "...왜?"

아무도 답하지 않는다.

---

멀리서 — 아주 멀리서 — 바람인지 목소리인지 모를 속삭임.

*"쉬어도 되지 않겠나?"*

캄만 희미하게 고개를 돌린다. 더스크의 빛이 완전히 꺼진다.
카이는 속삭임을 못 들었다. 메르카도, 하크도, 미아도 못 들었다.

진은 — 들었을지도 모른다. 글림이 한순간 평소보다 짙어진다. 진이 카이의 손을 잡는다.

진: "익숙해지면 안 된다고 했지? 이건 그래서야."
카이: "...이해 못 하겠어."
진: "나도."

카이가 피즈를 안는다. 피즈의 빛이 약하지만 — 꺼지지는 않았다.

**— 시즌 1 끝 —**

## 제6부: 시즌 2로의 씨앗

시즌 1에서 심어진, 아직 설명하지 않은 것들 — **"씨앗이다"를 본문에서 말하지 않는다.** 각 장면은 그 자체로 완결되어 있고, 나중에 다시 보면 더 의미가 드러난다.

| 씨앗 | 심어진 지점 | 나중에 쓸 시점 |
|------|-----------|-------------|
| 빛이 안 나오는 거래 | 에피소드 12 | 시즌 3+ |
| 메르카의 "나를 통해" 대사 | 에피소드 8 | 시즌 2 다크 미들러 아크 |
| 메르카가 카이에게 거절당한 순간의 인상 | 에피소드 11 | 시즌 2 |
| 메르카에게 던져진 "공짜로 해줘?" 발언 | 에피소드 5 | 시즌 2 |
| 캄의 "그때는 나도 저랬어" | 에피소드 3 | 시즌 3 고요파 |
| 캄의 점점 꺼지는 더스크 | 전 시즌 | 시즌 3 |
| 고요의 속삭임 (캄만 들음) | 에피소드 12 | 시즌 2 엔딩~3 |

| 노움의 떨림 (리브라의 한계) | 에피소드 6, 12 | 시즌 4 리브라 아크 |
| 시그니스가 팹을 바라봄 | 에피소드 1, 9, 12 | 시즌 6+ 해글의 열쇠 |

| 벨의 경매장 장면 (가지는 순간 식음) | 에피소드 6 | 시즌 2~3 그리드 에그/욕망 아크 |
| 마르코가 볼트를 본 순간 | 에피소드 2 | 시즌 3 결빙 예언 |

## 부록: 시즌 1 톤 체크리스트

- [ ] 이면 질문("내 일에 의미가 있는가")을 직접 말하는 대사 없음
- [ ] 모든 갈등에 양면 — 하크도 미아도 옳음. 메르카도 진심이다.
- [ ] 서프라이즈: 모든 규칙을 지킨 거래에서 빛이 안 나옴 (예상 못한 결말)
- [ ] 적대 세력 직접 등장 없음 — 씨앗만 심음
- [ ] 고요의 자 목소리는 시즌 마지막 장면, 한 문장만
- [ ] 팹이 뒤를 돌아보지 않는 장면 최소 3회
- [ ] 글림의 미스터리에 답을 주지 않음
- [ ] 조용한 클라이맥스 — 폭발이 아니라 "빛이 안 나오는" 적막
- [ ] 일반인 눈 vs 딜러 눈 대비 장면 최소 3회 (에피소드 1 할머니, 에피소드 3 진의 지적, 에피소드 11 거래 관중)
- [ ] 카이가 능동적 선택을 하는 장면 최소 2회 (에피소드 8 그리드 에그, 에피소드 11 초보 지원)
- [ ] 캄의 의미 있는 장면 3회 (에피소드 3, 7, 12)
- [ ] "이 장면은 나중을 위한 복선"을 본문에서 말하지 않음
- [ ] 현실 공간(전자상가, 석조 창고 등)과 루멘 지역(볼트랜드 등)의 겹침을 보여주되 과설명하지 않음
- [ ] 메르카를 "속으로는 악당"으로 연출하지 않음 — 시즌 1의 메르카는 진짜 좋은 사람
- [ ] 벨의 첫 등장(에피소드 6): 경매장에서 "가지는 순간 빛이 꺼지는" 장면 한 번. 대사 없음. 설명 없음
- [ ] 볼트의 첫 등장(에피소드 2): 마르코가 멀리서 바라봄. 볼트는 고마운 보호자로 묘사됨. "결빙의 전조"는 본문에서 말하지 않음
- [ ] 크러스트가 움직이지 않는다는 사실이 묘사되되, 그 의미를 설명하지 않음
