# EP12 — 빛이 안 나오다 (시즌 피날레)

> *"...왜?"*

**핵심**: 모든 규칙을 지킨 거래에서 빛이 안 나온다. 시즌 1의 조용한 클라이맥스. 고요의 속삭임.
**감정 곡선**: 일상 → 평범한 거래 → [적막] → 혼란 → 각자의 반응 → 속삭임 → 침묵
**페이지**: 약 38~44p (시즌 피날레 분량)

## 시퀀스 A — 평범한 하루 (p1~6)

### 페이지 1

**[P1-1] WIDE — 전자상가, 오후**
평범한 오후. 카이가 거래 준비 중. 피즈가 어깨에. 불규칙한 깜빡임이 이제 일상이 됐다.

**[P1-2] 카이, 앱을 봄**
매칭된 구매자와 약속 장소. 평범한 중고 전자기기 거래.

**[P1-3] 카이의 표정**
담담하다. 긴장도 흥분도 없다. 프로 같은 건 아니고 — 그냥 익숙해진 거다.

> **카이 (독백)**: *열다섯 번째 거래.*

### 페이지 2~3 — 거래 과정

**[P2-1] 중간 컷 — 구매자 도착**
구매자(30대 남성). 평범하고 좋은 사람. 인사. 악수.

**[P2-2~P3-4] 거래 과정 6컷 — 꼼꼼하게**

카이는 메르카에게 배운 대로, 진에게서 배운 대로, 자기가 겪으며 배운 대로 — 완벽하게 거래한다.

- 상대 조건 먼저 듣기 ✓
- 물건 상태 투명 공개 ✓ (작은 스크래치까지 보여줌)
- 가격은 시세 확인 후 양쪽 합의 ✓
- 둘 다 이해한 상태에서 동의 ✓
- 강제 없음 ✓
- 속임 없음 ✓

> **카이 (독백)**: *투명하게. 공정하게. 자유롭게. 리브라의 세 룬.*

## 연출 노트

### 전체
- **시즌 피날레의 핵심은 적막**: 폭발이 아니라 부재. 빛이 안 나오는 순간 모든 배경음이 사라지는 연출
- **SPLASH(p5)**: 시즌 1에서 가장 중요한 1컷. "아무것도 안 일어났다"를 전면으로. 빈 공간의 힘

### 고요의 속삭임
- **1문장만**: "쉬어도 되지 않겠나?" 이것만. 추가 설명 없음
- **캄만 듣는다**: 다른 모든 캐릭터는 못 들음. 진이 감지했을 가능성만 암시
- **위협이 아니다**: 고요의 자는 위협하지 않는다. 위로한다. 그래서 무섭다
- **더스크가 꺼지는 것은 폭력이 아니라 수긍**: 캄이 스스로 멈추는 것. 자유교환의 원리를 위반하지 않는다

### 시즌 2로
- 이 에피소드 이후 독자의 질문: "빛은 왜 나는가?" → 시즌 2의 표면 질문 "탁한 빛도 빛인가?"로 자연스럽게 연결
