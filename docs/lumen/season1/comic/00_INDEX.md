## 컷 표기법

```
[P1-1]  = 페이지 1, 컷 1
[P1-2]  = 페이지 1, 컷 2
WIDE    = 가로 와이드 컷 (풍경, 분위기)
TALL    = 세로 풀 컷 (인물 등장, 임팩트)
SPREAD  = 양면 펼침 (클라이맥스)
INSET   = 작은 삽입 컷 (표정 클로즈업, 디테일)
SPLASH  = 전면 1컷 (강렬한 한 장면)
BLEED   = 테두리 없이 페이지 밖으로 빠짐
SFX     = 효과음
```

## 연출 원칙

- **현실 톤**: 일상적 색감, 구체적 배경 (전자상가, 골목, 카페)
- **루멘 톤**: 채도 높고 빛 입자 흩날림. 현실 배경 위에 반투명으로 겹침
- **가시성 연출**: 일반인 시점 컷에서는 버디가 반딧불이/빛 반사로만 보임
- **감정 = 빛**: 캐릭터 감정이 강할수록 버디의 빛이 밝거나 어두워짐
- **침묵 컷**: 대사 없는 컷을 두려워하지 않는다. 표정과 빛만으로 전달
