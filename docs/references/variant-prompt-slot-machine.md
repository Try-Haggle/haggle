# Variant Prompt — "The Jackpot Is Closer Than You Think"

> A slot machine experience. Every pull shows a different platform eating your money.
> The user keeps pulling, getting frustrated. Then — Haggle lands. Jackpot.
> The journey from loss to win IS the story.

## Design Notes for Variant

**Visual progression across pulls:**
- Pull 1: Machine pristine, red flash = "oh that's not great"
- Pull 2: Machine cracking, deeper red = "this is getting worse"
- Pull 3: Machine worn, orange flash = "the system is rigged"
- Pull 4: Green glow builds, JACKPOT = "everything transforms"

**Key animation moments (describe in prompt if Variant supports motion):**
- Reels spinning with motion blur
- Coins pouring from tray
- Machine restoring/healing on jackpot
- Loss counter flipping to savings counter
- Lever transforming into CTA button

**Color journey:**
- Pull 1-3: increasingly red/dark tones (#f43f5e → #be123c → #881337)
- Pull 4 buildup: green glow creeps in (#10b981)
- Jackpot: explosion of green + gold (#10b981 + #fbbf24)
- Post-jackpot: clean, confident dark + green (#0f172a + #10b981)

**Typography:**
- Machine labels: monospace (JetBrains Mono)
- Headlines: bold sans-serif (Inter 800)
- Receipts: monospace, left-aligned, receipt-paper texture
- "JACKPOT": largest text on the page, glowing, possibly animated scale-up

**The emotional arc:**
Curiosity → Disappointment → Frustration → Hope → EUPHORIA → Understanding
"Oh cool a slot machine" → "ouch" → "ouch again" → "wait what's that green" → "YES" → "oh, Haggle is always the jackpot. I get it."
