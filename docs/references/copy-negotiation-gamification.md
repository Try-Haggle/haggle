## DESIGN NOTES FOR AI TOOL

**Overall feel:**
- Dark mode primary (navy/black) with accent glows (blue for buyer, green for seller/Haggle)
- Game UI influences — health bars, round counters, win/loss indicators
- But NOT childish — think "Bloomberg Terminal meets Duolingo". Professional gamification.
- Motion: price numbers tick up/down, progress bars fill, cards flip on reveal
- Typography: bold, large numbers. The money you save should be the biggest thing on screen.

**Color associations:**
- Haggle/positive: #10b981 (emerald green)
- Buyer actions: #3b82f6 (blue)
- Seller actions: #f59e0b (amber)
- Competitor fees/loss: #f43f5e (rose red)
- Neutral/background: #0f172a (slate 900)

**Key interactive moments:**
1. Hero calculator — type a price, watch fees animate in real-time
2. Negotiation arena — animated offer/counter-offer bouncing
3. Strategy picker — hover to see stats change
4. Fee split slider — card tilts physically (already built in fee-calculator-v2)
5. Trust Score — progress bar that fills on scroll reveal
