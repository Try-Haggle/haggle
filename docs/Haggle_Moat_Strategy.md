# Haggle 해자 전략 (Moat Strategy)

*Last Updated: 2026-03-30*

## 1. 구조적 해자 6개

**이해충돌** — Google은 광고, Shopify는 판매자 수익과 충돌. 협상은 적대적이라 양쪽을 동시에 섬기기 어려움.

**분쟁 해결** — 아무도 안 하고 싶어하는 일. 판례 축적이 시간 자산. 협상-분쟁이 묶여 분리 운영 불가.
- 구현 현황: `dispute-core` 패키지 완성 (99 tests). DS 패널 투표, 가중치 티어(BRONZE~DIAMOND), K-Means 폴백, 보상 계산까지 구현됨.

**중립성 증명** — 온체인 기록으로 공정성 사후 검증. 광고 수익 없는 중립 인프라만 가능.
- 구현 현황: ABI 정의 완료 (`contracts` 패키지). Solidity 미작성, 컨트랙트 주소 null. `payment-core`에서 viem 클라이언트 준비됨.

**카테고리 깊이** — NIG + CategoryConfig로 도메인별 최적화. 범용 협상으로는 불가.
- 구현 현황: 미구현. `ExpertiseDomain` enum (electronics/luxury/fashion/collectibles/automotive/general)만 존재.

**양면 에이전트 네트워크** — 구매/판매 에이전트 네트워크 효과 + 전환 비용(평판, 데이터, 구독).
- 구현 현황: `trust-core` + `commerce-core`에 신뢰점수, 콜드스타트, SLA 패널티, 전문성 배지 구현됨. 에이전트 평판/전환비용 로직은 미구현.

**규제 선점** — Legal Opinion Letter, 주별 라이선스 선점.
- 구현 현황: 코드 범위 외 (법률 업무).

## 2. Skills Marketplace = 핵심 해자

### 철학: "인증의 마켓플레이스"

Haggle이 인증을 직접 하지 않고, 인증 서비스를 거래에 연결하는 플랫폼. StockX(자체 검수)와 근본적으로 다른 책임 구조.

### 수익 구조

| 수익원 | 비중 | 설명 |
|--------|------|------|
| 거래 수수료 1.5% (캡 없음) | ~84% | Skills가 고가 시장을 열어서 극대화 |
| 플랫폼 수수료 30% | ~13% | 서드파티 Skill 서비스에서 커미션 |
| API 마크업 | ~3% | 원가보다 높게 청구, 소규모 차익 |

무료 Skill(시리얼, IMEI, NFC 등)은 직접 수익 $0이지만 거래 신뢰를 높여 성사율을 극대화.

- 구현 현황: 설계 문서만 존재 (`docs/engine/16_스킬_마켓플레이스.md`). SkillManifest, SkillCoordinator 등 코드 0줄. **MVP에서 LegitApp API 연동을 첫 번째 Skill로 구현 예정.**

## 3. 인증 파트너 리서치 결과 (2026-03-30 업데이트)

### MVP 파트너 결정: LegitApp ⭐️

LegitApp을 MVP 유일한 인증 파트너로 확정. 이유:

1. **커버리지 최대**: 11개 카테고리, 300+ 브랜드 — 스니커즈/핸드백/시계/트레이딩카드/수집품 모두 커버
2. **신뢰도 압도적**: Trustpilot 4.7/5 (250+리뷰, 90% 5점), App Store 4.9/5 (8,600+평가)
3. **API 확인됨**: REST API + 웹훅 (developer.legitapp.com), 비즈니스 인증 후 접근
4. **정확도**: 99.9%+ (2020년 이후), 3백만 건 인증, 200만 사용자
5. **확장 가능**: 하나의 API로 전 카테고리 순차 확장 가능

#### 가격 상세

| 카테고리 | 표준가 | 30% 할인 (대량) |
|----------|--------|-----------------|
| 스니커즈 | $3 | $2.10 |
| 트레이딩카드/피규어 | $4 | $2.80 |
| 스트리트웨어/코스메틱 | $4 | $2.80 |
| 명품 핸드백 | $10~$20 | $7~$14 |
| 에르메스 | $20+ | $14+ |
| 시계 | $15+ | $10.50+ |

- 25% 할인: $500+ 단일 구매, 30% 할인: $2,500+ 단일 구매
- 월 2,000건+: 커스텀 프라이싱
- 처리 시간: 스니커즈 ~12시간 (30분 패스트트랙 옵션), 핸드백 30분~3시간
- 일 처리 능력: 3,500건

### 파트너 비교 (전수 조사)

| 서비스 | 방식 | 브랜드 | 카테고리 | 시계 | API | Trustpilot | 가격/건 | MVP 적합성 |
|--------|------|--------|----------|------|-----|------------|---------|------------|
| **LegitApp** | 사진+AI+전문가 | 300+ | 11개 | ✅ | ✅ REST+웹훅 | **4.7/5** (250+) | $3~$15 | **채택** |
| CheckCheck | 사진+AI+전문가 | 365+ | 7개 | ❌ | ✅ 기업용 | 3.5/5 (28) | $0.76~$1.33 | 저가격이나 신뢰도 부족 |
| Entrupy | **디바이스+AI** | ~22 | 3개 | 핑거프린트만 | ❌ 하드웨어 필요 | 3/5 | $7~$28 | **P2P 불가 — 탈락** |
| Legit Check | 사진+전문가리포트 | ~30 | 3개 | ✅ | ❌ 없음 | 혼합 (130) | $9.99 | API 없음 — 탈락 |
| Real Auth | 사진+전문가 | 170+ | ~8개 | ✅ | ❌ 미확인 | 4.0/5 (162) | 미공개 | API 미확인 — 탈락 |

#### 탈락 사유 상세

- **Entrupy**: 물리 현미경 디바이스 필수. P2P 판매자가 핸드폰으로 사진 제출 불가. 물리 매장 전용 솔루션.
- **Legit Check**: Enterprise API 없음. 소비자 구독 서비스. B2B 연동 불가.
- **Real Auth**: REST API 미확인. 12~24시간 소요. 시계 인증서 위조 문제 보고됨.
- **CheckCheck**: 가격 저렴하나 Trustpilot 3.5/5 (28건 — 표본 부족), 시계 미지원, 핸드백 최근 추가(2025.4)로 정확도 미검증.

#### 전자기기 한계

5개 서비스 **모두** 일반 전자기기(스마트폰, GPU 등) 인증 미지원. Phase 0~1 전자기기 카테고리는 IMEI/시리얼 자체 검증으로 대응 필요.

### 사진 기반 인증의 한계 (솔직한 평가)

모든 사진 기반 서비스는 외관만 확인 가능. 시계 무브먼트, 금속 성분, 무게 등은 불가. 하지만 여러 레이어를 쌓으면 각각의 약점을 보완하고, 에스크로가 최종 보호.

## 5. NFC 검증 (2020년+ 롤렉스)

Haggle 앱에서 NFC 카드 직접 읽기 → gcard.rolex.com 검증 → 시리얼 OCR 자동 매칭.

- 비용: $0, 개발 2~3주
- 증명하는 것: 카드 진품 + 시리얼 일치
- 증명 못하는 것: 시계 자체의 내부 진위 (별개 문제)
- 무료 기본 제공으로 리스팅 신뢰도 향상 → 거래 성사율 ↑

다른 브랜드: Omega/Patek은 Extract of Archives ($150, 8~12주 — P2P 비현실적), Breitling은 블록체인 디지털 여권. 2026년부터 업계 전체가 NFC/블록체인 표준화 추세.

## 6. VideoAuthSkill — 시장에 없는 것 (선점 기회)

RVI 기술(Blitzz/SightCall)은 성숙하지만 럭셔리 인증에 적용한 서비스는 **전무**. Haggle이 최초.

- 감정사가 영상 통화로 실시간 지시 (줌인, 프리즈 프레임, AR 마크업)
- 사진보다 많은 확인: 크라운 작동, 소리, 야광, 베젤 회전, 무게(저울)
- 전체 세션 자동 녹화 → 분쟁 시 최강 증거
- 비용: $30~$80, 감정사는 집에서 하루 10~15건 가능
- 시계뿐 아니라 명품백(지퍼 작동), 차량(시동 소리), 전자기기(작동 확인)에도 적용

## 7. 분쟁 시스템과 Skill 연동

> 구현 현황: DS 패널 투표 엔진 완성 (`dispute-core`, 99 tests). 가중 투표, 티어별 보상, K-Means 폴백, 증거 검증, 상태머신 모두 구현됨. API 라우트 9개 등록. **Skill 결과 → 증거 연동은 Skills 시스템 구현 후 진행.**

**원칙: Skill 결과 = 증거, 판정 = DS 패널 투표**

- Skill 등급이 높다고 자동 승리 ❌ (실물 감정이 AI보다 떨어질 수도 있음)
- DS 패널이 모든 증거를 종합하여 투표로 판정
- Skill 등급은 참고치일 뿐

**분쟁 중 추가 Skill:**

1. 분쟁 접수 → 72시간 기본 증거 수집
2. "추가 인증 요청 기간" → 양쪽이 원하는 Skill 추가 가능
3. Skill 결과 나올 때까지 시간 자동 연장 (최대 +7일)
4. 비용은 요청한 쪽 부담, 승소 시 패소 쪽 전가 옵션

**Skill 충돌 시:** DS 패널이 모든 증거를 보고 종합 판단. 가중치 피라미드는 참고.

## 10. Product Passport

1회 인증 → Entrupy Fingerprint + 거래 이력 + 인증 이력 → Base L2 온체인 기록.

- 재거래 시 재인증 불필요 (3초 Fingerprint 매칭)
- 이력 쌓일수록 제품 가치 ↑
- Haggle 생태계 Lock-in (다른 플랫폼에서 거래하면 체인 끊김)
