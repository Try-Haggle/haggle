# 게이미피케이션 설계 v4

> 거래 활동을 통해 고유한 버디를 수집하고 성장시키는 시스템.
> "좋은 거래를 할수록 좋은 버디" — 양쪽 모두에게 공정한 보상 설계.

**구현 상태:** 🔧 구현 중 (DB/API 완료, HC 경제 업데이트 중)
**선행 조건:** trust-core, dispute-core 완성 (✅) → DB/API Phase 3 완료 (✅)

## 1. 핵심 컨셉

```
거래 완료 (DEAL) → 양쪽 모두 에그 1개 획득 (무료)
에그 자동 부화 → 버디 등급 독립 확률 결정
대표 버디 장착 → 거래 시 버디 XP 획득
HC (Haggle Credits) 경제로 소비·보상 순환
```

**설계 원칙:**
- **양쪽 공정**: 구매자·판매자 동일 보상. 한쪽 "승리"를 보상하지 않음
- **좋은 거래 보상**: 분쟁 없이 상호 만족한 거래일수록 더 큰 보상
- **비대칭 능력 배제**: 상대방 패턴 읽기, 양보 확률 조작 등 불공정 능력 없음

**스택: Python (마스크 색칠) + CSS (애니메이션) + DB (저장)**
**라이브러리 0, .riv 없음, NFT 없음 (MVP)**

## 3. 에이전트 레벨 (XP)

### XP 획득

| 행동 | XP | 비고 |
|------|-----|------|
| 거래 완료 | +100 | 구매든 판매든 (양쪽 동일) |
| 거래액 보너스 | +amount^0.4 × 5 | $100=+40, $1K=+100, $10K=+250 |
| 연속 성사 스트릭 | +50 | 3건 연속부터 |
| 배심원 판정 | +30 | 1건당 |
| 분쟁 없이 거래 | +20 | 보너스 |
| **상호 긍정 평가 (양쪽 4+)** | **+20% XP** | **양쪽 모두에게 적용** |
| **일방 긍정 평가 (받은 쪽)** | **+10% XP** | **받은 쪽에게만 적용** |

**XP 기본 공식:** `baseXP = 100 + amount^0.4 × 5`
**XP는 절대 감소하지 않음. 레벨도 내려가지 않음.**

### 등급 (6단계 + 각성)

| 등급 | 테두리 | 특징 |
|------|--------|------|
| ⬜ Common | 흰색 | 기본 색상 |
| 🟩 Uncommon | 초록 | 약간 특별한 색 조합 |
| 🟦 Rare | 파란 | + 무늬 (줄무늬/물방울/별) |
| 🟪 Epic | 보라 | 특수 색상 (홀로그램/네온/메탈릭) |
| 🟨 Legendary | 금 | 전설 전용 디자인 (투명/황금/우주) |
| 🔴 Mythic | 홀로그램 | 종족당 3종, 총 24마리 한정 |
| 🌟 Awakened | 고유 | 등급 체계 초월. 고유 이름·문양. 세계에 하나 |

> **등급 = 탄생 시 결정 (확률) + 진화로 성장 가능 (천장 있음) + 각성으로 초월 (예외적)**
> 자세한 진화 규칙은 아래 "버디 진화 (Evolution)" 참조.

### 버디 진화 (Evolution)

버디는 대표 버디로 장착한 상태에서 거래를 완료하면 BLv(버디 레벨)이 오르고,
일정 레벨에 도달하면 **외형이 진화**하며 현재 등급이 한 단계 상승한다.

**핵심 규칙:**
- 탄생 등급에 따라 **진화 천장**이 결정됨
- 천장 이상으로는 일반 진화로 올라갈 수 없음
- 천장을 초월하는 유일한 방법은 **각성 (Awakening)**

```
탄생 등급             진화 천장 (일반 경로 최대)
──────────────────────────────────────────
Common / Uncommon  →  Epic까지
Rare / Epic        →  Legendary까지
Legendary          →  Mythic까지
Mythic             →  Mythic (각성만 남음)
```

**진화 트리거 (BLv 기준):**

| 진화 | 필요 BLv | 대략 거래 수 | 비고 |
|------|---------|------------|------|
| → Uncommon | BLv.10 | ~40건 | 외형 미세 변화 |
| → Rare | BLv.25 | ~100건 | 고유 무늬 출현 |
| → Epic | BLv.50 | ~200건 | 특수 색상·오라 |
| → Legendary | BLv.80 | ~350건 | 전설 디자인 |
| → Mythic | BLv.120 | ~550건 | Legendary 탄생만 가능 |

> **이미 해당 등급 이상으로 태어났으면 해당 진화 단계는 건너뜀.**
> 예: Rare로 태어난 버디는 BLv.25 진화를 건너뛰고 BLv.50에서 Epic으로 진화.

**진화 시 변화:**
- 외형: 등급에 맞는 시각적 업그레이드 (종족별 도감 참조)
- 테두리: 새 등급 테두리로 변경
- 능력: Legendary 진화 시 태그 능력 풀에서 2개 획득

**각성 (Awakening) — 천장 초월:**
- 천장에 도달한 버디가 추가 조건 충족 시 등급 체계를 초월
- 탄생 등급과 무관 — Common 출생도 각성 가능
- 각성 조건: 거래 횟수 + Trust(Glicko-2) + 분쟁 없는 연속 거래 + HC 순환 이력
- 각성 버디: 고유 이름, 고유 문양, 세계에 하나뿐, 멘토 전환 가능
- Common 출생 각성 = Legendary 출생 각성보다 **더 희귀하고 더 어려움**

### 에그 드롭

```
거래 완료 (DEAL) →  구매자: 에그 1개
```

## 5. 등급 확률 시스템

### 고정 확률 (COMMON ~ EPIC)

| 등급 | 확률 | 비고 |
|------|------|------|
| ⬜ Common | 55% | 고정 |
| 🟩 Uncommon | 28% | 고정 |
| 🟦 Rare | 13% | 고정 |
| 🟪 Epic | 3.9% | 고정 |

### 동적 확률 (LEGENDARY, MYTHIC만)

```
adjusted = base × ln(1 + amount/100) / sqrt(1 + totalMinted/5000)
```

| 등급 | base | 비고 |
|------|------|------|
| 🟨 Legendary | 0.097% | 동적 감소 |
| 🔴 Mythic | 0.01% | 동적 감소, 총 24마리 한정 |

### 정규화

```
6개 등급 확률 합산 → 100%로 정규화

예시 ($500 거래, totalMinted=2,000):
  COMMON 55%, UNCOMMON 28%, RARE 13%, EPIC 3.9%
  LEGENDARY adjusted = 0.097% × ln(1+500/100) / sqrt(1+2000/5000) ≈ 0.15%
  MYTHIC adjusted = 0.01% × ... ≈ 0.015%
  → 6개 합산 후 100%로 정규화
```

### MYTHIC 캡

```
MYTHIC 총 수량: 태그당 6마리 (종족당 1~2마리), 총 36마리
해당 태그 MYTHIC 전부 출현 → 해당 태그 MYTHIC 확률 → 0
전체 36마리 다 나오면 → MYTHIC 확률 → 0
```

## 6. 천장(Pity) 시스템

### 개요

일정 거래 누적 시 상위 **탄생 등급** 에그 보장. **볼륨 기반 OR 품질 거래 횟수** 중 먼저 도달하는 조건 발동.
**Pity 발동 시 누적 카운터(볼륨·품질 거래 횟수 모두) 0으로 리셋.** 다시 처음부터 쌓기 시작.

### Pity 조건

| 등급 | 볼륨 기반 | 품질 거래 횟수 | 비고 |
|------|---------|------------|------|
| 🟪 Epic | $3,000 누적 | 15 품질 거래 | 먼저 도달 시 발동 |
| 🟨 Legendary | $15,000 누적 | 40 품질 거래 | 먼저 도달 시 발동 |
| 🔴 Mythic | — | — | **Pity 없음** (순수 행운 + 고액 거래) |

### 품질 거래 (Quality Trade) 정의

```
다음 조건을 모두 충족하는 거래:
  1. 거래 금액 ≥ $20
  2. Clean Deal (분쟁 없음)
  3. 상호 별점 ≥ 3점 (양쪽 모두)
```

### 볼륨 추적

```
$20 미만 거래: Pity 볼륨에 불포함
$20 이상 거래: "유효 기여금" = 거래금액 × 품질 배수(QM)
품질 배수(QM)가 적용된 금액이 누적됨
```

## 7. 품질 배수 (Quality Multiplier)

### 개요

좋은 거래 습관에 대한 보상. Pity 볼륨 누적 속도에 영향.
**양쪽 모두 동일한 배수 적용** (공유 거래 지표 기반).

### 배수 항목 (곱셈 적용)

| 조건 | 배수 | 비고 |
|------|------|------|
| 분쟁 없음 (Clean Deal) | ×1.3 | 분쟁 0건 |
| 효율적 협상 (≤60% of max rounds) | ×1.2 | 빠른 합의 |
| 상호 긍정 평가 (양쪽 3+) | ×1.2 | 양쪽 만족 |
| 연속 딜 3회+ | ×1.2 | 거래 연속성 |
| Trust 80+ | ×1.1 | 높은 신뢰 |

### 설계 원칙

```
✅ 포함: 양쪽이 공유하는 거래 지표만
❌ 제외: 절약률 배수 — 비대칭 (구매자 유리), 공정성 위반

예시: Clean Deal + 효율 협상 + 상호 평가 = 1.3 × 1.2 × 1.2 = ×1.872
$100 거래 → 유효 기여금 $187.2
```

## 11. 버디 조합 (Fusion)

### 규칙

```
같은 등급 버디 3마리 → 해당 등급 이상 탄생 등급 에그 1개 보장

비용: HC 없음 (버디 3마리가 비용)

의미: 조합 에그의 탄생 등급이 높을수록 진화 천장이 높음
      → 더 높은 천장의 버디를 얻는 수단
```

### 조합 결과

| 입력 | 출력 | 비고 |
|------|------|------|
| Common × 3 | Common+ 에그 | Uncommon 이상 가능 |
| Uncommon × 3 | Uncommon+ 에그 | Rare 이상 가능 |
| Rare × 3 | Rare+ 에그 | Epic 이상 가능 |
| Epic × 3 | Epic+ 에그 | Legendary 가능 (극히 낮은 확률) |
| Legendary × 3 | Legendary+ 에그 | Mythic 가능 (극히 낮은 확률) |

- 에그 드롭: **거래 보상** (가챠 아님) → FTC 대상 아님
- 확률 공개: **필수** (한국/FTC 대비)
- MVP에서 NFT 없음: DB 기반 → 규제 대상 아님
- 버디 교환: HC 기반 P2P → 스마트 컨트랙트 필요 시 별도 검토
- 추후 Soulbound: "활동 보상 기념품" → 증권 아님
- 용어: **"NFT"라 부르지 않음** → "Digital Collectibles" 또는 "버디"
- HC: 플랫폼 내 보상 포인트 → Money Transmitter 아님 (환불 불가)
