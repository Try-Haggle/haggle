## 4. 게이미피케이션 (확정)

### 캐릭터 시스템
- **8종족** (16카테고리 그룹화)
- **마스크 기반 레이어 합성** — Python 자동, 0.065초/캐릭터
- **CSS 애니메이션** — 라이브러리 0, .riv 불필요
- 크로스 드롭 15% (다른 종족)

### 캐릭터 등급 (6단계)
Common → Uncommon → Rare → Epic → Legendary → **Mythic**

### 확률: 거래액 연속 함수 + 동적 감소
```
rate = base × ln(1 + amount/100) / sqrt(1 + total_minted/5000) × (remaining/max)
```
- 거래액 높으면 확률 ↑ (연속, 분기 없음)
- 총 민팅 늘면 확률 ↓
- 해당 등급 소진되면 확률 → 0

### 조합 수: ~1,613억+ (중복 사실상 불가능)

### 뽑기 타이밍
- 매 거래 → 일반 뽑기 1회
- 새 파라미터 조합 → 보너스 뽑기
- 새 카테고리 → 종족 확정 뽑기
- 마일스톤 → 등급 상승 뽑기
- 배심원 10건 → 전용 뽑기
